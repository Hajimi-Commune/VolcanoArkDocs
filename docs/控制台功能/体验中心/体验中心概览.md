# 体验中心概览
您可以在体验中心快速试用不同模型的能力，评估模型性能，找到最适合业务场景的模型。体验中心提供友好的交互界面，支持选择模型、版本和调用方式，并可快速配置模型参数。
在使用语言类模型时，还可以启用联网、MCP 服务器、Canvas 和知识库等工具，进一步提升并延展模型的能力。

体验中心支持试用以下模型：

* 语言模型：执行文本理解和视觉理解任务。
* 视觉模型：执行图片、视频和 3D 生成任务。
* 语音模型：执行语音识别、语音合成、实时语音交互等任务。
* 向量模型：执行图、文向量检索任务。

此外，您还可以在体验中心试用精调后的模型和智能路由推理接入点。
