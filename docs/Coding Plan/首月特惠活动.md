# 首月特惠活动
方舟 Coding Plan 是专为 AI 编码打造的订阅套餐，首购首月最低仅需 9.9 元，即可在主流 AI 编码工具中使用最新的 Doubao-Seed-Code、DeepSeek-V3.2 模型。本文为您详细介绍活动相关信息。
## 活动细则
### 活动时间
2025.11.11 - 2026.05.11
### 活动对象
所有完成认证（包含个人认证和企业认证）的火山引擎用户，每个实名认证主体可享受一次首月优惠价。
### 购买入口
[单击前往活动](https://www.volcengine.com/activity/codingplan)，了解套餐规格及定价。
### 首次订阅优惠规则

* 订阅单月套餐：享受首月优惠价。
* 订阅多月套餐：仅有 1 个套餐享受首月优惠价，其余套餐原价。

### 活动规则

1. 购买限制：单个账号最多可以购买 6 个月。
2. 购买不同套餐：
   * 同一账号只能购买一个类型的套餐，请根据使用需求选择合适的套餐。
   * 同一时间只有一个套餐生效，当前套餐失效后，下一个套餐会自动生效。
3. 回购规则：如果当前有生效的套餐则不能回购。
4. 续费规则：
   * 在下单页选择自动续费：即连续包月，每次固定续费 1 个月，到期会自动扣除下一个月的套餐费用。
   * 在下单页没有选择自动续费：套餐到期后会短信通知用户，用户需要重新订阅套餐。
   * 在下单页手动续费：再次购买已订阅的套餐，将根据购买的月数延长订阅的有效期。
5. 升配规则：
   * 有效期不变，补齐要升配套餐的差价后即可升配。
   * 仅支持升配，不支持降配。
6. 退款规则：请注意套餐购买后不支持退款，您可以在当前周期内使用服务。建议您根据使用需求选择合适的订阅套餐和周期。

## 常见问题
### Coding Plan套餐优惠是首月优惠还是首次购买优惠？
不论是订阅一个月还是多个月的套餐，都仅是首月享受优惠价格，再次购买及续订都无法享受该优惠。
### 购买Coding Plan套餐，享受优惠有什么限制？
一个认证主体只能享受一次套餐优惠。如果该认证主体下有多个账号，则只有首次购买的账号能享受该优惠。
### 我购买时为什么没有优惠？
如果您没有享受首购优惠，需要您确认：当前账号的认证主体是否有其他账号已经购买过Coding Plan套餐。
### 我购买了Lite套餐，还能再购买Pro套餐吗？
不能。同一账号只能购买一个类型的套餐，但是Lite套餐可以通过补足差价升级为Pro套餐。
如果您首次购买Lite套餐，在首月期间可按Pro套餐首月优惠价补足差价升配。升配补差价规则，参见[升配规则](https://www.volcengine.com/docs/6269/438722#.5Y2H6YWN)。
### **如何升级我的套餐？**
您可以通过访问方舟 Coding Plan 订阅活动或方舟控制台进行套餐升级操作。升级后立即生效。
### 套餐的用量额度限制是多少？
方舟 Coding Plan 提供 2 种订阅套餐，满足不同开发场景的需求。

- 套餐 | 适用场景 | 用量限制
- Lite 套餐 | 中等强度的开发任务，适合大多数开发者。 | * 每 5 小时：最多约 1,200 次请求。
- * 每周：最多约 9,000 次请求。
- * 每订阅月：最多约 18,000 次请求。
- Pro 套餐 | 复杂项目开发，适合高强度工作的开发者。 | > Lite 套餐的 5 倍用量。
- * 每 5 小时：最多约 6,000 次请求。
- * 每周：最多约 45,000 次请求。
- * 每订阅月：最多约 90,000 次请求。

注意：实际用量会因开发任务复杂度、代码库大小及 Coding 工具的使用方式等因素有所不同。
### 支持使用哪些模型？
支持使用最新版 Doubao-Seed-Code、DeepSeek-V3.2 模型。
### Coding Plan 支持哪些编程工具？
Coding Plan支持多款主流编程工具，如Claude Code、Cursor、Cline、Codex CLI 等编程工具。您使用的编程工具会共用您所订阅套餐的额度。
### Coding Plan 套餐如何接入编程工具？
需要使用 Coding Plan 专用 API Base URL 接入，接入步骤可参考[接入AI编程工具](/docs/82379/1928262)。

* 配置 Base URL（如Claude Code）：`https://ark.cn-beijing.volces.com/api/coding`
* 兼容OpenAI API工具Base URL（如Cline、Cursor、Codex CLI）：`https://ark.cn-beijing.volces.com/api/coding/v3`

注意：套餐额度仅在支持的 Coding 工具中生效，不能用于 API 调用，API 调用需要按照方舟现有计费规则进行收费。
### 套餐额度耗尽后是否会继续消耗其他资源包或账户余额？
不会。套餐额度在时间周期内耗尽后，您需要等待下一个周期自动恢复额度，不会消耗其他资源包或账户余额。

* 每 5 小时限额：根据实际请求发生时间，滑动 5 小时窗口动态刷新。
* 周限额：每周一 00:00:00 刷新。
* 月限额：每订阅月第1日 00:00:00 刷新。

### **我可以同时在多个工具中使用我的订阅套餐吗？**
可以，您可以在所有支持的工具中使用同一订阅套餐，但额度是共享的，所有工具的使用会消耗同一套餐额度。
### 订阅套餐后可以退款吗？
订阅服务一经购买即视为确认，不支持退款。即使您未使用完套餐，费用也无法退回。我们建议您根据使用需求选择合适的订阅套餐和周期。
### 如何取消订阅自动续费？
您可以在订阅管理页面取消自动续费。请务必在下一个扣费日至少 3 天前取消，避免自动续费。取消自动续费后，当前周期继续有效，但到期后会不再续费。
### 套餐是否支持团队使用？
目前方舟 Coding Plan 主要面向个人开发者。团队协作使用，请使用方舟模型 API 按量付费方式接入。
### 套餐是否仅可用于编码工具中的调用？
是的。单独调用 API 是独立计费的，不可享用 Coding 套餐的额度。在非 AI Coding 产品中使用方舟 Coding Plan 权益对应的 Base URL 和 API Key 有可能被我们视为滥用和违规，导致订阅停用或账号封禁。
