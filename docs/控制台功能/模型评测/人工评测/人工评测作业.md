# 人工评测作业
# 1. 上线任务
	

任务创建者在 **我创建的** tab下，找到待上线任务，点击 **是否上线** 按钮，将任务上线。
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_5bdf5c9a9b448101dfec86cba8ab2102.png)
# 2. 标注员身份任务作业
	

人工评测任务页面，切换至 **我评测的** tab, 可以查看以标注员身份需要作业的评测任务。
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_be82b8fa61bcf24405d074d78c0b5a25.png)
点击待评测的任务名称，点击右上角 **开始答题** 按钮，进入到评测页面。
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_e34b84e67865aa5f1d85e3d08385e461.png)
针对多模型或者单模型模式的模型效果进行评测标注。专业版模式下，可以评测GSB分数，diff问题类型，各个模型的有用性，真实性，安全性，自身分体类型等内容，点击右上角 **提交** 按钮，提交本题并且领取下一道题。
专业版多模型评测页面
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_3afa7749281911bde301fe431f00cc1e.png)
专业版单模型评测页面
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_c3759d47031beca88491d72a5ddbffa9.png)
 # 3. 审核员身份任务作业
	

人工评测任务页面，切换至 **我审核的** tab, 可以查看以审核员身份需要作业的评测任务。审核标注员标注的每一道题。
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_b731ce1fddd128b66d985f9e47b7ef05.png)
点击待评测的任务名称，点击右上角 **开始答题** 按钮，进入到评测页面。
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_447082cb59a930ac4233372e4041cf42.png)
评测页面与标注员的评测页面相同，可以对在 **审核备注** 处填写审核意见。可以直接修改标注员标注的评测结果，也可以点击右上角的 **驳回** 按钮，将此题驳回给标注员进行重新标注。可以点击右上角的 **通过** 按钮，通过此题，同时自动领取下一道题。
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_1e0e0ce4cb39f9ff390d0562946b3d9c.png)
# 4. 质检员身份任务作业
	

人工评测任务页面，切换至 **我质检的** tab, 可以查看以质检员身份需要作业的评测任务。
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_ec049aa48c92cb0876c1bba68ae135be.png)
点击待评测的任务名称，点击 **质检抽检** tab。
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_5a6d4d71a6e5cfc015513e4fb42cd8bc.png)
点击左上角 **创建抽检包** 按钮。
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_27d4bb206ed745afba845d35e5e3e21a.png)
可以选择优先抽检的模式，并且自定义抽检题目数量，点击 **确定** 按钮创建抽检包。
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_d369951bc7dd0fe6cb5ecb7804a8c189.png)
在质检抽检列表页面，点击抽检包的 **开始抽检** 按钮，点击某一道试题的 **抽检** 按钮进行质检作业。作业页面与评测标注页面相同，可以填写 **质检备注** 内容对质检评论进行备注
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_6067ab34fe0af975af68657367809c0f.png)
所有试题抽检完成后，点击抽检包的终止按钮，终止质检状态。
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_4109c00b0ccd8e7f0018293447bfae5c.png)
点击 **批量操作** 按钮，选择是否将任务推送至验收状态，或者驳回所有试题。
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_188215f8722dd12cde0c7983a2cf644a.png)
 # 5. 验收员身份作业任务
	

人工评测任务页面，切换至 **我验收的** tab, 可以查看以验收员身份需要作业的评测任务
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_6ab457054751864542ea283a5c41059c.png)
点击待评测的任务名称，点击 **质检抽检** tab
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_5d0a925d46b536207f3a53a8ccf2698c.png)
点击左上角 **创建抽检包** 按钮，创建一个抽检包
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_3e9dd944679bba26426b5285d57ebe54.png)
可以选择优先抽检的模式，并且自定义抽检题目数量。
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_2c5f954b88f65976bc9503dfecdf502b.png)
在质检抽检列表页面，点击抽检包的 **开始抽检** 按钮，进行质检作业。作业页面与评测标注页面相同。
在 **我验收的** tab, 设置任务的合格状态或者不合格状态。
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_7fba1770913a6dae405b0c74685df606.png)
