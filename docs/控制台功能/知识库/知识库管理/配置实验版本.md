# 配置实验版本
本节将说明如何配置知识库的实验版本。
:::tip
实验版本可以帮助您在现有知识库文档的基础上，配置不同的离线处理策略（如：嵌入模型、切片策略等），并测试其效果。这一过程，您不需要再次上传文档。
:::
# 操作步骤

1. 实验版本功能入口。实验版本目前有两个功能入口，分别如下方示意图所示。

![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/2831e373458d4e23b98f46cdf02567b6~tplv-goo7wpa0wc-image.image =2940x)
![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/918316c09fcc47eb9fc41cb61a4388e2~tplv-goo7wpa0wc-image.image =2940x)

2. 新建实验版本。

实验版本配置与创建知识库几乎一致，您可在创建实验版本时配置您想要测试的文档处理策略。以下说明几点需注意的配置项：

   1. 实验版本名称：不可与正式版本重名。以英文字母开头且不能为空，只能包含字母、数字及 _，长度为 1 ～ 128 个字符。
   2. 数据类型：知识库实验版本的数据类型必须与主版本保持一致。
   3. 同步新增文档：开启时，在知识库正式版本上传的文档会自动在当前实验版本策略上同步处理。
   4. 其他参数的设置您可参考[知识库参数说明](https://www.volcengine.com/docs/84313/1254463#:~:text=%E9%85%8D%E7%BD%AE%E7%B1%BB%E5%88%AB,%E5%8F%82%E6%95%B0%E8%AF%B4%E6%98%8E)。

**注：每一个实验版本的资源隔离，会独立出账计费，请谨慎创建！**
![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/a7c9596c3ebc41d69d05a21732212b3c~tplv-goo7wpa0wc-image.image =2940x)

3. 实验空间打开。

可以看到正式版本和实验版本的名称，点击后可切换版本或者配置不同版本。
导入文档并解析处理完成后，即可进行知识检索和知识问答。
![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/ff7fea71bf2d425087f701f6477ab3a2~tplv-goo7wpa0wc-image.image =2940x)

4. 导入文档。

**注：实验版本不支持上传文档，仅支持从正式版本下进行文档的导入。**
可选择导入正式版本下的部分文档进行测试，也可选择一次性导入正式版本下的全部存量文档。
![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/f62215680be54385bf8d9e9a05c364e2~tplv-goo7wpa0wc-image.image =2940x)

5. 发布为正式版本。

您可选择将实验版本发布为正式版本，发布后当前实验版本的配置策略将显示到最外层的知识库，原正式版本对应的配置策略将默认作为实验版本为您保留。
**注：仅实验空间下的文档是知识库主版本的全量文档时，该实验空间才可被发布为正式版本。**
![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/d67a94ccdd3448a780356af6e5779a91~tplv-goo7wpa0wc-image.image =2940x)

6. 删除实验版本。

您可在知识库版本信息中选择编辑或者删除实验版本。
![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/7e97e6e39b5d46c493dc4ef470ca7431~tplv-goo7wpa0wc-image.image =2940x)
