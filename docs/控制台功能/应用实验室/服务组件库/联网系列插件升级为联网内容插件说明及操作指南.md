# 联网系列插件升级为联网内容插件说明及操作指南
火山方舟大模型服务平台的联网插件（基础版、plus版、pro版）不同版本之间存在价格差异和产品能力差异，增加了使用门槛。因此，我们将联网插件（基础版、plus版、pro版）合并升级为联网内容插件。

# 插件升级说明
新版联网内容插件进行了**全方位升级**，丰富内容源并实现链路优化，与最新豆包大模型配合使用效果提升。[立即开通](https://console.volcengine.com/ark/region:ark+cn-beijing/components?action=%7B%7D)。**原联网基础版、联网plus版、联网pro版插件**将于**2024年12月24日18:00**起**停止服务**，不再支持调用。我们强烈建议您，提前规划升级执行计划，在旧插件下线前参照有关指引完成**新旧插件升级**，保障业务运营不中断。如有任何问题，请随时联系我们。

# 新旧插件对比
新版插件核心变化：

* **回答效果优化**：经评测零代码应用回答效果提升
* **搜索资源调整**：默认展示联网资源的url；所有资源可按需选择是否搜索，不再根据版本区分搜索资源
* **默认限流提升**：默认限流由1 QPS提升为5 QPS，支持工单提交QPS提升需求
* **调用费用调整**：联网资源由免费调整为每月2万次免费额度，超出部分按4元/千次计费

# 插件升级操作
在进行插件升级前，需先开通联网内容插件：

* **方式一**：进入[服务组件库](https://console.volcengine.com/ark/region:ark+cn-beijing/components?action=%7B%7D)，点击【开通】按钮，开通联网内容插件。

![Image](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_0a38a78821d3e09db60217882a33d01f.png =5952x)

* **方式二**：进入应用实验室，点击【立即开通】按钮，开通联网内容插件。

![Image](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_70763fa773bb5d8dff679721e4abf71d.png =5952x)

* **方式三**：进入应用实验室，进入【应用编辑页】，点击【立即升级】按钮，开通联网内容插件。

![Image](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_314a84f9a75f0f029a57112270080c44.png =5952x)
如已开通联网内容插件，进入应用实验室，进入【应用编辑页】，点击【立即升级】按钮并确认，完成从旧版联网插件（基础版、plus版、pro版）到新版联网内容插件的升级。
![Image](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_a24a4b8bdb84e2d43ed4017c6e1f6363.png =8966x)
