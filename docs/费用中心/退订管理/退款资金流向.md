# 退款资金流向

## 退款资金流向

预付费退订或降配会出现退款，相关资金流向说明：

- 自2023年11月7日之后支付购买的实例服务，付款后6个月（180天）内发生退订或者降配，退款金额一般原路退回。即：通过支付宝、微信或网银支付等三方支付渠道的云产品订单，退款金额会退至支付使用的支付宝、微信或网银账号等原三方支付渠道，通过火山账户余额支付的订单会退至账户余额。
- 在2023年11月7日之前支付购买的实例服务，若仅进行了新购下单购买，并未进行续费、更配等其他订单类型的下单交易，则退款金额也支持原路退回。
- 除了上述情况以外，退款金额会退至您的火山账户余额。

> **说明**
> 若对应的订单已开过发票，退款金额将退至火山账户余额。您可以通过账户余额提现的方式将款项提现到您的支付渠道；相关提现操作请参考[提现操作指引](../资金管理/提现操作指引.md)；

- 若产品退订成功或资源降配成功，退款金额大约会在2个工作日内退回。退款仅指用户以现金方式支付的订单金额（包含信控额度），不包含代金券抵扣的部分。
- 若您使用信控额度完成支付，则在退款后也将原路返还至信控额度中。
- 退还后，相关金额去向可前往收支明细中进行查看。
- 相关退订规则请参考[退订规则说明](退订规则说明.md)，相关降配规则请参考[资源变更计费规则说明](资源变更计费规则说明.md)。
