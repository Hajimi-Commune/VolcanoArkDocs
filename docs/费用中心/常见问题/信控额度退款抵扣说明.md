# 信控额度退款抵扣说明

【信控额度退款抵扣】是指当用户使用了火山引擎提供的信控额度进行消费后，若发生退款、更配或调账的情况，这部分因退款而返回的金额称为信控额度退款。信控额度退款的金额不会直接退回到用户的现金账户中，而是存放在专门的信控额度退款账本里，并自动优先用于冲销【历史消费时原订单的账单费用】。

例如用户在某个月份有信控的消费并且当月发生了退款，这部分退款优先会用来抵扣您当月购买时的账单费用，而不是变成可提现的现金余额。

特殊情况：当信控额度退款用于冲销【历史消费时的原订单的账单费用】时，如果原订单已经被现金冲销还款，那么该笔信控退款则转化为现金返还至您的账户余额。

注意事项：

- 信控退款优先用于冲销历史月份的原消费账单，可能导致所涉及被还款月份账单的未结清金额与可开发票金额不一致。
- 建议始终依据账单中的【应付金额】字段进行开票和付款。
