# 分账账单

## 功能简介

由于部分云产品的出账计费粒度不等于资源管理粒度，所以需要使用分账账单功能将其计费金额拆分至资源维度，分账账单中分拆项ID等于云产品的资源粒度ID。

即产品的账单用量由各分拆项用量综合产生，账单金额也会根据分拆项用量情况进行自动拆分。

例如字节跳动大模型服务（豆包大模型）的费用账单是按照模型维度（输入）（输出）计费，但是资源管理上是按照EndPoint接入点（下文简称EP）进行管理，分账账单进行拆分时会根据不同的EP用量对金额进行拆分，对于豆包大模型而言，分账账单中的分拆项ID对应的就是EP资源。

类似的CDN的流量费用由各"Domain域名"产生、对象存储的存储费用由各"bucket存储桶"产生、共享带宽包的带宽费用由各"EIP公网IP"产生，其中Domain、bucket、EIP等被称为该产品的分拆项。

分账账单包含了这类用量型产品及其他普通产品的账单明细，您可在分账账单查看各个分拆项的具体用量/用量占比和分拆费用，以此作为企业内部分账的参考依据。

> **注**：如您拥有多个账号并建立了财务托管关系，请分别登录各账号开通查看分账账单数据。

## 操作说明

### 1. 开通分账服务

进入"[账单管理-分账账单](https://console.volcengine.com/finance/bill/split-bill/)"

> **注**：开通后，新生成的账单明细将生成对应的分账账单明细（开通前的存量账单明细无对应分账数据），数据最晚存在2天延迟；

### 2. 查看分账账单明细数据

完成开通后，进入分账账单页面，查看分账账单明细数据，部分重要字段说明如下：

| 字段名称 | 解释说明 |
| --- | --- |
| 账务账期 | 代表分账账单数据的归属月份 |
| Payer账号 | 代表账单的支付方，付款方 |
| Owner账号 | 代表账单的归属方，使用方 |
| 产品 | 云产品的商品名称，例如：字节跳动大模型服务（豆包大模型），对象存储，内容分发网络 |
| 实例ID | 计费维度的ID，包含实例ID和实例名称 |
| 计费单元 | 计费项中的计费单元，例如：Doubao-pro-32k推理（输入），标准存储容量费用，流量 |
| 分拆项id | 资源粒度的ID，如豆包大模型的EP，TOS的Bucket，CDN的Domain等 |
| 分拆项名称 | 分拆项的用户自定义名称 |
| 用量 | 对于分拆型产品，指分拆项的具体用量；如cdn带宽费用的计费方式为月95峰月结时，分拆项的分拆用量均统计该月内各域名带宽的月95峰值作为分账依据，若为按流量月结，则分拆用量为该月内各域名流量值 |
| 分拆用量占比 | 指每个分拆项的用量在总用量中的占比；按账期汇总时无该字段 |
| 其他字段 | 可见[新版账单管理说明文档](新版账单管理.md)中账单明细处说明 |

用户可以将资源（分拆项）放置在所需的项目中，也可以在资源（分拆项）上标记不同的资源标签。在之后的账单数据分析中，就可以根据项目维度和标签维度进行账单数据统计和管理。

**1）以方舟字节跳动大模型服务（豆包大模型）为例**

首先进入方舟控制台，在左上角选择自己要创建资源的项目，这一步决定了接下来创建的资源的项目归属，也会影响有哪些IAMUser可以访问该资源。

然后点击【自定义推理接入点】Tab，选择【创建推理接入点】按钮

然后填写所需业务信息，并且绑定相关的用户业务标签，例如这是用于玩具业务的测试环境资源，然后选择其他需要的配置信息后，点击确认接入

最终创建好了带有标签和项目信息的大模型接入点endpoint

**2）对象存储TOS为例**

进入对象存储控制台，创建存储桶时可以将桶选择到项目中，并点开高级设置，标记标签信息

**3）以内容分发网络CDN为例**

进入CDN控制台，新增域名时可以将域名选择到项目中

> **注**：若删除资源或者将标签与资源解绑，则新增的分账账单中标签信息为空，请谨慎操作；

### 4. 分账账单数据查看

**1）按分拆项ID查看账单金额**

**2）按项目、标签查看账单金额**

通过上方的筛选条件，筛选不同项目的账单数据，筛选不同标签的账单数据。如果想按照项目统计金额，可以点击右上方的【查看项目汇总分布】按钮

### 5. 获取分账账单数据

您可通过页面导出、账单自动存储TOS、分账账单OpenAPI获取分账账单数据；

**1）页面导出**

点击"导出账单CSV"，按需选择导出分账账单数据；

**2）账单自动存储TOS**

可见[账单存储TOS](../账户总览/账单存储TOS.md)说明文档，添加订阅项时可按需选择对应的分账账单；

**3）分账账单OpenAPI获取分账账单数据**

接口ListSplitBillDetail

可见[账单OpenAPI说明文档](../API参考/账单管理/ListSplitBillDetail.md)；调试可跳转[API Explorer](https://api.volcengine.com/api-explorer/?action=ListSplitBillDetail&groupName=%E8%B4%A6%E5%8D%95%E4%B8%AD%E5%BF%83&serviceCode=billing&version=2022-01-01)
