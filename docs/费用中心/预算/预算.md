# 预算

## 功能简介

预算可以帮助您进行云成本规划、监控和管理。

您可以创建多个预算，并基于预算设置提醒，及时跟踪和了解预算执行情况。

## 操作说明

### 新建预算

可以创建多个预算，实现不同业务场景下的预算控制。

- **设置预算名称**：预算名称需唯一
- **设置预算范围**：根据需要控制预算的范围，设置过滤项（如账号、产品等）
- **设置预算信息**
  - **预算周期**：可选月/季度/年，均按照自然日规则在月初/季度初/年初0点重置数据
  - **起止时间**：整个预算的覆盖时间范围，在该范围内，可以有多个预算周期（按预算周期设置）
  - **预算计划**：可选预算周期内固定金额或按不同金额
  - **预算金额**：在一个预算周期内的预算金额；当选择固定金额时，每月/季度/年的预算金额相同；当选择按月/季度/年规划时，可在每个预算周期内填写不同的预算金额

### 设置提醒

根据创建的预算，设置提醒，便于了解和跟踪预算执行情况。

- **设置提醒阈值**：根据实际产生的账单应付金额，可以设置多个提醒阈值
  - **按预算比例**：按照预算金额的比例，达到该比例时触发提醒
  - **按固定金额**：按照固定金额，达到该金额时触发提醒
- **设置提醒消息接收人**：可以选择提醒被触发时的消息接收人和接收方式

注意：您可以在消息中心-接收人管理，管理接收人信息；信息修改后预算提醒内的接收人信息将在T+1日更新；

### 查看预算详情

可以查看详细的预算执行情况，包括完整预算周期内的预算金额、实际账单应付金额，差额与预算实现比例等。

您可以选择重新编辑该预算的相关信息（预算范围、提醒阈值等），预算数据将被重置（T+1日更新）；

## 规则说明和注意事项

- 每个用户最多可以创建100个预算；
- 预算监控每小时执行1次，因此当您接收到预算提醒时，实际成本可能已经超出预算，请合理规划您的预算；
- 同一消息接收人一天最多接到30条提醒；
- 在2次监控之间，若您的账单应付金额同时触达多个提醒阈值，则按最高阈值仅触发一次提醒消息通知；
- 若用户调整提醒阈值，则仅当金额再次触达新阈值时，将触发提醒消息；
