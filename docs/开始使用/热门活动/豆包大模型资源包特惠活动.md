# 豆包大模型资源包特惠活动
火山引擎豆包系列大模型推出若干款资源包特惠活动，购买资源包可享受比按token计费更低的折扣。本文为您详细介绍活动相关信息。
## 秒杀专区资源包
秒杀资源包包含豆包大模型1.6 资源包、Doubao-Seedance-1.0-pro 资源包、DeepSeek-R1资源包。
### 秒杀专区豆包大模型1.6 资源包活动规则
**活动时间**：**2025.6.11-2025.12.31**
**资源包有效期：3个月**
**活动对象**：通过火山引擎**实名认证的新用户**。
**活动模型**：Doubao-Seed-1.6、Doubao-Seed-1.6-thinking、Doubao-Seed-1.6-flash 支持**通用抵扣3款模型共1000万 tokens**。
**使用范围**：资源包仅支持抵扣**调用目标大模型进行在线推理**消耗的输入/输出 tokens。
**购买入口**：[单击此处](https://www.volcengine.com/activity/ark)，了解售卖的模型资源包规格及定价。
**活动规则**：
1.新客须知：
a.新客定义：对于豆包模型系列商品，未产生需支付订单或账单的用户。
b.对于秒杀专区豆包资源包和企业/个人专区的新客专享部分的资源包，**新客仅可享受一次新客价，若再次购买，则会恢复原价**。
c.下单成功任一新客专享价的资源包或下单成功后于7天内申请退订，都视为消耗您的新客身份，您无法再次享受新客活动优惠，请谨慎操作。
2.活动商品每日限量抢购，抢完为止，次日恢复额度。
3.活动商品为特价商品，不支持叠加使用代金券结算。活动页面价格仅供参考，请以结算时的最终成交价格为准。购买成功后可前往 [火山方舟](https://www.volcengine.com/experience/ark?model=deepseek-r1-250120) 进行调用。

8.活动页面价格仅供参考，请以结算时的最终成交价格为准。
9.不支持抵扣缓存命中的token费用。
10.成功购买资源包后，可于 [费用中心->资源包管理](https://console.volcengine.com/finance/resource-package) 处查看资源包余量等状态信息。
**特别说明**：
1.多个账户通过同一手机号、同一证件号、同一账号ID，或其他经火山引擎排查多个账户为同一实际控制主体的，均视为“同一用户”。
