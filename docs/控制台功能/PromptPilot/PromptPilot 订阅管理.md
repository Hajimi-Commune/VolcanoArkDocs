# PromptPilot 订阅管理
本文为您介绍 PromptPilot 订阅、续费、增购积分、退订、团队管理相关的操作步骤。
# 功能入口

1. 登录 PromptPilot 平台，在主界面左下角点击个人头像，在弹出的菜单中单击 **套餐订阅**，即可进入订阅套餐界面。
2. 右上角可切换 **订阅套餐**、**当前套餐**、**积分详情**、**团队管理**（仅团队版支持）页面。

![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/6547a7c5ca0645c39e460ef9fedc5f5b~tplv-goo7wpa0wc-image.image =2826x)

# 套餐订阅
用户登录默认为免费版套餐，如果想要享受更高的权益，可以在 **订阅套餐** 页面，订阅个人标准版和团队版套餐。
![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/8918c52e33d84fbc9ef4a0d53f89a7f7~tplv-goo7wpa0wc-image.image =2958x)

# 套餐续费
套餐支持自动续费。如需手动续费，在 **当前套餐** 页面，单击 **立即续费** 即可。
![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/d68a0d5430154c31835d9a37b9b1fc73~tplv-goo7wpa0wc-image.image =2330x)

# 增购积分
只有付费套餐有效期内可以增购积分，详细的积分抵扣规则参见 [积分资源包说明](https://www.volcengine.com/docs/82379/1807500?lang=zh#%E7%A7%AF%E5%88%86%E8%B5%84%E6%BA%90%E5%8C%85)。
如需增购积分，在 **当前套餐** 页面，单击 **增购** 即可。
![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/7e8c362b6b1b492daa8821ec92702746~tplv-goo7wpa0wc-image.image =2066x)

# 团队管理
当您订阅团队版套餐成功后，在主界面左下角点击个人头像，可切换至团队空间。

## 创建团队
首次进入，需创建您的团队空间，以方便后续进行团队成员管理和积分使用管理等操作。 
<div style="text-align: center"><img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/6212a877fa3a47368f4c5191ffff33b2~tplv-goo7wpa0wc-image.image" width="400px" /></div>

## 添加成员
团队版超过10个成员将会收取超额成员费用，套餐外成员单价 = 660积分/人/天。

1. 单击 **添加成员**，填写待邀请成员的 **账户ID** 和 **用户ID**（在被邀请者的左侧下角头像信息中获取）点，并选择成员的权限。
2. 单击 **发送邀请**，完成成员添加。

# 退订
你可以通过火山引擎费用中心退订管理页面操作退订，仅支持退订套餐，增购的积分包不支持退订。

1. 打开火山引擎费用中心[订单管理](https://console.volcengine.com/finance/order)页面，找到需要退订的订单，点击 **查看详情** ，复制页面下的 **实例ID**。

<div style="text-align: center"><img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/9b9d7376286249e7b3045da587c96adf~tplv-goo7wpa0wc-image.image" width="3434px" /></div>

<div style="text-align: center"><img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/1d26c2ed910b40f3a9a974b1c0cd5273~tplv-goo7wpa0wc-image.image" width="3590px" /></div>

2. 切换到 [退订管理](https://console.volcengine.com/auth/login?redirectURI=/finance/unsubscribe) 页面，单击 **非七天无理由退订**，输入步骤 1 复制的实例 ID 进行搜索。
3. 单击 **退订**，进行退订操作。
