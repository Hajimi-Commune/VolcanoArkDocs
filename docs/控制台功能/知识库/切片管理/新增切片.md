# 新增切片
本节将说明如何在知识库中增加新的切片内容。
# 操作步骤
在知识库详情页中的【处理结果】Tab下，可针对某一原始文档新增切片内容。
![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/5d56f505ea754152b3ee419adf04a520~tplv-goo7wpa0wc-image.image =2313x)

点击【新增切片】按钮，新增切片内容。
:::warning
新增切片会对切片embedding，并产生费用。
:::

* 非结构化文档
   * 所属文档：选择原始文档，将新增的切片追加到对应的文档内
   * 切片内容：填入新增的切片内容

![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/3b7b1f510ae2435ca044726c27527036~tplv-goo7wpa0wc-image.image =2319x)

* 结构化文档
   * 所属文档：选择原始文档，将新增的切片追加到对应的文档内
   * 表头字段：在每个字段下填入新增的内容

![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/d1e372705fef47c48df9dde5ecdd2b81~tplv-goo7wpa0wc-image.image =2556x)

* 非结构化FAQ文档
   * 所属文档：选择原始文档，将新增的切片追加到对应的文档内
   * 问题：新增切片对应的问题
   * 答案：新增切片对应问题的答案
