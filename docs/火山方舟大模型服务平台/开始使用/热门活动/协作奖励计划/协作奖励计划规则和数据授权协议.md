# 协作奖励计划规则和数据授权协议
<h1 id="0d8e2dea"><strong>协作奖励计划规则</strong></h1>

火山引擎邀请您参与豆包模型（“专有模型服务商”）协作奖励计划（“本计划”），本计划旨在与您一起共建更优质的模型服务。
## **协作奖励计划权益说明**
本计划合作期（2024年5月15日至2024年11月30日）内，在您首次订购豆包模型时如选择参与本计划，火山引擎将为您提供5亿 tokens的**豆包奖励资源包（“奖励包”）** 权益，此奖励包您无需另行付费。同一用户仅可参与并领取一次奖励包。奖励包有效期至2025年3月31日，即您在合作期内领取的奖励包需要在2025年3月31日前使用完毕，否则将被系统回收清零。**您在奖励包额度内调用豆包模型服务期间向模型输入和模型生成的任何数据，包括但不限于所有输入的文本、声音、图形、图片、图像、视频、软件或任何其他类似信息（合称“客户数据”）提供给豆包模型，用于模型优化。**
作为新用户，您开通豆包模型服务时，火山引擎还将为您提供500,000 tokens**免费资源包，如您同时参与本计划，那么您免费资源包**额度内调用豆包模型产生的客户数据，将授权给豆包模型用于模型优化。如您不参加本计划，则您在免费资源包额度内调用模型产生的客户数据将不会提供给豆包模型。
## 数据授权说明
**本计划项下授权的客户数据为您参与本计划并开始消耗奖励包或免费额度期间你输入模型和模型输出的所有客户数据。您理解并确认，专有模型服务商对于授权的客户数据的许可使用期限为永久。若您已参与本计划，但在参与过程中希望撤回授权，您可以在当前页面点击“取消授权并退还奖励包”按钮，那么您后续的推理数据将不再被提供给模型方，且本奖励计划将自行终止，您的奖励包内未使用的资源将被释放。**
其他数据授权相关约定，详见[豆包模型数据授权使用协议](/docs/82379/1391984#051ad91f)。
## **特别说明：**
（1）如多个账户通过同一手机号、同一证件号、同一账号ID，或其他经火山引擎排查多个账户为同一实际控制人的，均视为“同一用户”。
（2）对恶意抢占活动资源、批量刷单及其他任何采用非常规手段影响活动公平性的行为，火山引擎有权取消从事违法违规行为的用户活动资格并回收奖励包资源，必要时追究违规用户的法律责任。
（3）火山引擎可以根据运营情况对活动进行变更或调整，包括不限于活动规则、商品数量等，请以页面最新内容为准。用户在活动规则变更或调整前已经获得的权益不受影响。

<h1 id="051ad91f"><strong>豆包模型数据授权使用协议</strong></h1>

本《豆包模型数据授权使用协议》（“**本协议**”）是您与**豆包模型**（“**专有模型服务商**”），就您授权专有模型服务商使用客户数据（定义见下文）的相关事项，签订的协议。
**【特别提醒】您在火山方舟大模型服务平台浏览、开通及使用的专有模型，由专有模型服务商提供。如您希望在授权专有模型服务商使用客户数据（定义见下文）时，您与专有模型服务商应采取相应安全措施保证客户数据传输安全，如需额外获得火山引擎的加密传输、隐私计算等附加技术服务的，您可以根据本协议第4条（通知）所载联系方式咨询火山引擎。**
**为了更好地保障您的权益，您在同意本协议之前，请您务必审慎阅读，并充分理解本协议的全部内容，特别是以加粗标注的条款。如果您对本协议的条款有任何疑问，可以根据本协议第4条（通知）所载联系方式向火山引擎询问，相关人员将向您解释条款内容。如果您不同意本协议的任意内容，或者无法准确理解相关人员对条款的解释，请不要在订购页面勾选同意。如果您在订购页面特别勾选了同意本协议，即表示您已接受本协议所述的所有条款和条件，届时您不应以未阅读本协议或者未获得相关人员对您问询的解答等理由，主张本协议无效，或要求撤销本协议。**
您确认，在您注册及使用本服务时，您应当是合法成立并有效存续的法人或非法人组织。如果您是中国大陆地区以外的用户，您订立或履行本协议还需要同时遵守您所属和/或所处国家或地区的法律。

## **1.定义与解释**
1.1 除非本协议另有约定，如下术语具有本款所规定的含义：
(1) **个人信息**：以电子或者其他方式记录的与已识别或者可识别的自然人有关的各种信息，不包括匿名化处理后的信息。
(2) **匿名化**：是指个人信息经过处理无法识别特定自然人且不能复原的过程。
(3) **关联方**：是指拥有一方、被一方拥有或与该方被同一其他方拥有的任何法律实体，其中“拥有”是指拥有相关实体超过50%的所有权或拥有指导其管理的权利。
(4) **火山方舟大模型服务平台**：是指由北京火山引擎科技有限公司（“**火山引擎**”）开发的，在机器学习平台上提供的一项服务功能，提供大模型的训练、推理、评测、精调等全方位功能与服务，并重点支撑大模型生态，火山方舟通过稳定可靠的安全互信方案，保障模型服务提供方的模型安全与模型使用者的信息安全。
(5) **客户数据**：指您在使用专有模型服务商提供的专有模型时，向专有模型输入和专有模型生成的任何数据，包括但不限于所有输入的文本、声音、图形、图片、图像、视频、软件或任何其他类似信息。
## 2.数据授权
2.1 授权目的：本规则项下的授权将用于开发机器学习、人工智能相关技术和豆包模型的优化、开发、使用等之目的。
2.2 授权范围：您同意，您将授予豆包模型一项**全球范围内的、非独家的、不可转让的、不可分许可的（但可分许可给专用模型服务商关联方以及为达成授权目的第三方外包服务商）、不可撤销的、免费的**权利，允许模型服务商为达成授权目的而**传输、存储、使用、复制、下载、修改或以其他方式处理**客户数据。
2.3 授权期限：本协议项下授权的起始日期为您**在订购页面勾选同意接受本协议之日**。授权期限为**永久**。如您在同意接受本协议后，（i）拟对该授权期限进行变更或调整，或（ii）如您希望停止对专有模型服务商授权新客户数据的，您可以通过点击【终止授权】立即终止您对被授权方的授权（**“终止操作”**）。**您进一步充分知悉、理解并同意，即便您完成了终止操作，但由于机器学习和人工智能技术的特殊性，一旦（i）您授权专有模型服务商使用相关客户数据，且（ii）相关客户数据已被使用的，则对该部分客户数据的使用将在技术上无法撤回。专有模型服务商仅能根据您的要求停止使用您新输入的客户数据。**
2.4 **您特别承诺并保证，您已合法持有客户数据，并已获得客户数据的一切必要充分授权或原始权利，以确保（i）您依法有权向模型服务商及其分许可方提供该等数据，且（ii）模型服务商及其分许可方有权根据本规则处理您提供的客户数据。若您发现您的客户数据出现任何权利争议或潜在权利争议，应立刻以书面形式根据本协议第4条（通知）所载联系方式，通知模型服务商，并尽您最大努力协助模型服务商处理该等争议。如因您违反本条约定和/或任何第三方针对模型服务商及其分许可方在本协议约定范围内使用客户数据提起任何主张，您应当赔偿模型服务商及其分许可方因此遭受的所有损失。**
2.5 个人信息：您应当避免向模型服务商提供您的个人信息。如果您提交给模型服务商的客户数据中不可避免地含有个人信息的，**您应当保证（i）该等个人信息的来源符合所适用的法律法规的规定，尤其是中国大陆地区法律法规的相关规定，并且（ii）您已获得了相关个人信息主体和/或实体的同意（包括必要情形下的单独同意、书面同意）和/或授权，且该等个人信息主体和/或实体授权同意的范围足以确保您能够根据本协议向模型服务商提供该等个人信息，并确保模型服务商有权在本协议第2.2及2.3条所述授权范围及授权期限内处理该等个人信息**。任何情况下，客户数据将在火山方舟平台的严格加密下，通过安全方案匿名化处理后提供给豆包模型。
2.6 保密数据：在任何情况下，您不应违反任何法律法规的规定或您与任何主体的约定，向模型服务商提供任何保密信息或商业秘密。
2.7 其他特殊数据：如您拟向模型提供商提供任何其他特殊数据的（例如，国家机密、核心数据、重要数据、人类遗传资源信息、地图数据等），**除遵守本协议第2.4条以外，您在提供前应当已获得相关政府部门一切必要充分的审批、备案或许可，并以书面形式根据本协议第4条（通知）所载联系方式，向火山引擎提出申请**，申请书内应当载明（i）该等特殊数据的类型（无需说明任何数据内容），（ii）已获得的政府部门审批、备案或许可，（iii）提供该等特殊数据的目的及必要性，以及其他火山引擎可能需要了解的情况。**在确保不会访问您拟提供的特殊数据内容的前提下，火山引擎有权对于您是否可以在火山方舟大模型服务平台上与模型服务商发生此等特殊数据的交互，作出特别决策。**
2.8 **特别说明，您参与火山引擎协作奖励计划的情况下，您需遵守该计划相关数据授权和使用规则。**

## **3.使用要求与限制**
3.1 使用要求。在本协议第2.3条所述授权期限内，您对专有模型服务商在本协议项下的授权，**仅应通过火山方舟大模型服务平台开展。未经火山引擎书面同意，您不得绕过火山引擎通过其他平台、渠道或方式向专有模型服务商提供任何类似于本协议之客户数据授权。**
3.2 被禁用途。专有模型服务商承诺，其不会将您的客户数据用于以下用途：（i）诽谤、侮辱、骚扰、跟踪、或威胁任何个人，侵犯或损害任何第三方的合法权利；（ii）生成、传播任何有害的、不适当的、亵渎的、淫秽的、有损他人权利的、色情的、错误的、欺诈的、侵权的、不良的、违法的、或其他有悖公序良俗的材料或信息；（iii）参与或鼓励任何可能导致您或火山引擎承担刑事或民事责任的行为；（iv）擅自以您或火山引擎的名义作出进行宣传、推广、商业活动、或做出引人误解或混淆的上述行为；或（v）进行违法活动。
3.3 被禁行为。您承诺，您不得（且不得尝试）：（i）对专有模型进行反向工程、解密、反向编译或反向汇编（本条款禁止但现行法律允许的除外）；（ii）绕开专有模型中的技术限制或其相关文献中的限制措施；（iii）移除专有模型中出现的任何著作权、商标、专利或其他标识；（iv）向第三方分发、再许可、出租、租赁或租借专有模型； （v）以开发竞争性产品或服务为目的使用专有模型；（vi）传输或上传任何包含病毒、木马病毒、蠕虫病毒、定时炸弹病毒、删除蝇、或任何其他类似程序的文件，其目的或用途在于破坏、摧毁、干扰或损害专有模型服务商或火山引擎或其他人的网络、计算机系统或其他设备；或（vii）进行违法活动。
