# 应用广场
应用广场从大模型可以落地的业务提效、企业经营效率提效两大场景切入，沉淀了深度推理、安卓终端、智能导购等多个垂直领域的 AI 原型应用。助力中小企业“一键复制”，完成基础应用搭建，后续还会持续引入更多不同类型、功能更丰富的应用，快速步入应用落地的实干阶段。
# 简介
火山方舟推出的[应用广场](https://console.volcengine.com/ark/region:ark+cn-beijing/application?projectName=undefined)以卡片式布局为主，页面上方有简明的**介绍**和[功能入口](/docs/82379/1528787#b34252e7)。广场中的**应用**都有独立的卡片展示区域，卡片上展示应用名称、简介、特性标签等关键信息。

# 应用卡片
## 场景介绍
在应用广场，点击**应用名称**进入场景介绍。
在场景介绍可以查看应用简介、架构图、代码说明、关联模型及云产品以及联系我们的方式。
![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/8fd880d412164285a5ea1aa66dc3d8db~tplv-goo7wpa0wc-image.image =3310x)
## 体验应用
大部分应用提供免费体验机会，无需复杂的付费流程，就能直接感受应用在实际场景中的表现。以“深度推理 DeepSearch” 为例，用户能够个性化配置最大问题拆解层数以及每次搜索最大关键词数。配置后输入复杂问题，观察其多轮思考与深度搜索的能力，深入了解应用能否为学术研究、企业决策等提供有效支持。

## Github 代码查看
火山引擎多个应用支持 GitHub 开源，提供丰富代码资源，助力开发者自由使用、修改并将其应用于各类场景，用户点击即可跳转Github源址。

## 复制应用
火山引擎多个应用支持便捷的复制功能，用户仅需简单操作，即可一键将应用复制为自身专属版本，灵活根据实际业务需求进行后续个性化定制与应用 。
