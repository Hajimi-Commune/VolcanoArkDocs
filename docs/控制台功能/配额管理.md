# 配额管理
控制台配额管理功能用于查看在线推理、批量推理、模型精调等模块的配额值与使用情况，并支持将配额分配至特定项目独占使用，实现资源使用情况的可视化与精细化管理。
## 配额管理

* **账号维度配额**

**账号维度配额**用于统一管理账号下所有项目的资源配额。具有权限（ArkFullAccess / ArkStandardGlobalAccess）的用户可进入方舟平台**账号全部资源**的页面，查看账号当前配额与使用情况，并以拖拽的形式进行项目配额分配。默认情况为所有项目共享账号配额，一旦某项目被分配了独占配额，该独占配额即成为该项目的硬性上限。

* **项目维度配额**

当选择特定项目时，将进入**项目维度配额**，项目成员可在此查看当前项目配额用量及共享配额中其他项目的使用情况。

## 配额权限控制
主账号默认显示账号下全部资源的配额情况，子账号默认显示默认项目（default）配额情况。权限控制通过[ IAM 策略](https://www.volcengine.com/docs/82379/1263493)实现，仅具备相应权限的用户方可执行账号配额查看、分配及管理等操作。推荐的配额权限策略如下：

- 策略 | 适用人员 | 说明
- **ArkFullAccess / ArkStandardGlobalAccess** | 平台管理员 / 项目负责人 | 拥有账号及所有资源的完整管理权限，可执行所有配额/限制相关操作。
- **ArkReadOnlyAccess** | 只读角色 | 仅具备项目配额与限制的只读权限，不允许进行任何修改操作或提交配额申请、创建告警。
