# 新版账单管理

## 账单出具规则

1. 请于每月第2自然日12点后查看上月账单数据，若存在计费模式=合同计费的账单时请于次月第4工作日查看上月完整消费。
2. 账单明细出具后，项目/标签参数最晚延迟12小时更新，获取CDN/TOS等开通型产品的分账数据请以"分账账单"为准。新建财务管理关系的子账号数据延后一天可查看。
3. 因云产品存在不同程度延迟出账，若操作退订/更配，操作前的产品账单可能延迟补充出账，退订后将不再产生新的计费。
4. 设置了提前自动续费的实例账单费用体现在发生续费的账期月份；
5. 财务管理主账号可查看已授权的财务管理子账号账单数据，发生新建/解除财务管理关系时，账单数据次日更新；
6. 2021年12月22日之后的账单数据已包含预付费（未包含更配、退订订单）和后付费的全部消费数据，2022年1月20日之后包含预付费更配、退订订单数据。
7. 自2021年8月1日起，账单明细费用支持到6位小数，汇总为账单后，抹掉3位及之后的小数，实际从账户扣费时按2位小数进行扣费（即扣到分）。在此之前出具的账单，账单明细以不足0.01元按0.01元取值为准。

## 账单扣款规则

1. 出具账单后，按照出账先后顺序进行扣款；
2. 优先抵扣代金券，超出部分使用账户余额抵扣；（代金券抵扣规则参考[代金券管理说明文档](../代金券管理/代金券管理.md)）
3. 用户充值后，默认按照出账先后顺序，进行未结清账单还款；

## 一、账单明细

### 1. 筛选条件

- **账务账期**：单选，默认为当月；代表账单归属的账期；
- **账号**：同时查询Payer账号和Owner账号的账单；
- **项目**：按项目模糊搜索；
- **订单号/账单号**：按订单号/账单号精确搜索；
- **实例ID/实例名称**：支持按实例ID/实例名称精确搜索；
- **统计项**：包含"计费项"、"实例"、"产品"、"账号"四个筛选条件；
- **统计周期**：包含"账期"、"按天"、"明细"三个筛选条件；统计项和统计周期选项默认为计费项+账期，此"账期"指"账务账期"；

### 2. 表格参数

- **账务账期**：代表账单出账归属账期；
- **业务账期**：代表业务实际发生期间；特别地，跨月调账等场景下，如3月调账1月账单，账务账期为3月，业务账期为1月；
- **日期**：统计周期为"按天"时可见该字段，后付费数据为消费结束时间的自然日，预付费为交易时间的自然日；
- **Payer账号**：代表此笔账单还款的对象；资源的支付结算账号；财务托管、代销\代售等场景下，Payer账号分别为托管主账号、代销\代售伙伴账号；
- **Seller账号**：代表售卖方；代销代售场景下，seller为服务提供商；
- **Owner账号**：代表真实使用服务的对象，即实际使用资源的账号；财务托管、代销\代售等场景下，Owner账号分别为托管子账号、代销\代售客户账号；
- **产品**：使用的产品/服务的商品名称；
- **业务类型**：业务开展的类型；包含普通业务、财务托管、财务管理、代销业务、代售业务等；
- **计费模式**：使用的产品/服务的计费方式，如包年包月、按量计费、合同计费等；其中合同计费为线下计费，您可联系商务了解计费详情；
- **消费时间**：预付费子订单的生效和到期时间、后付费结算周期的开始时间和结束时间；
- **使用时长**：订单的订购时长或按量付费的计费周期时长；
- **时长单位**：使用时长对应的单位；
- **交易时间**：消费记录对应的扣费时间，已出账未扣款时该参数为空；因后付费账单延迟出账，且月结账单最晚在次月第2个自然日12点生成，如次月2号12点前生成上月月账单并扣费，交易时间相比消费时间可能会发生跨月情况；
- **订单号/账单号/账单明细号**：预付费展示订单号、账单明细号，后付费展示账单号、账单明细号；
- **原订单号**：退订等场景下的原始订单号；
- **账单类型**：标记账单类型，如消费-使用、退款-退订、调账-人工等；
- **结算类型**：标记账单是否需要结算扣款，包含非结算、结算、quota结算；结算账单：用户正常使用产品\服务产生的费用账单，需支付结算，账单标记为结算；非结算账单：开通试用实例等场景下，无需支付结算，账单标记为非结算；quota结算：代销\代售业务场景下，终端客户使用云服务时，由伙伴同终端客户线下进行支付结算，账单标记为quota结算；
- **实例ID/名称**：账单对应的实例ID及实例名称；
- **配置名称**：实例/计费项的配置信息；
- **计费单元**：实例/计费项的计费单元信息；
- **地域**：实例/计费项的地域信息；
- **可用区**：实例/计费项的可用区信息；
- **影响因子**：实例/计费项的价格影响因子信息；
- **扩展字段**：实例/计费项的备注信息；
- **计费规则说明**：各个云产品的计费规则详细说明；
- **单价价格类型**：计费定价的价格类型，含固定单价、全额累进单价、超额累进单价、全额累进总价等；
  - 固定单价：不论用量多少，均使用此价格计算订单或账单金额；
  - 全额累进单价：按用量的绝对数额划分若干等级，对每个等级部分的数额分别规定相应的价格，就用量全部数额与之相适应的价格等级来计算账单金额。一定用量的数额只适用一个等级的价格；
  - 超额累进单价：当用量的数额超过某一等级时，仅就超过部分，按对应等级的价格计算账单金额，账单总金额是把每个等级超过部分的金额相加；
  - 全额累进总价：按用量的绝对数额划分若干等级，对每个等级分别规定相应的价格，当用量的全部数额落到等级之内时，收取这个等级设定的金额，即账单总金额等于设定的金额；
- **单价**：产品/服务的刊例价单价；特殊定价规则下，如阶梯定价，暂不提供单价；
- **单价单位**：单价的单位；
- **用量**：产品/服务的使用量；
  - 用量聚合逻辑优化，从2022-08开始的聚合逻辑如下：
    - 周期型计费项（此类账单费用与用量、使用时长有关，用量在下单时已确认），聚合时用量不会累加，比如1台云主机，每小时的账单中用量为1；按月聚合时用量仍为1；
    - 计量型计费项（账单费用仅与用量有关），聚合时用量会累加，比如文字识别在1-2点调用了100次，则用量为100，2-3点调用了100次，则用量为100，其他时间没调用；按月聚合时用量为200；
- **用量单位**：用量的单位；
- **资源包抵扣量**：产品/服务使用资源包抵扣的量；
- **预留实例券抵扣量**：预留实例本次抵扣的量；
- **解决方案名称**：将多种产品组合并进行二次开发的产品集成解决方案；
- **原价**：根据商品目录价计算出来的消费金额；部分线下合同计费账单，不体现优惠，即原价=折后价；
- **节省计划抵扣原价**：通过节省计划抵扣的原价金额；
- **优惠金额**：根据合同折扣、官网活动优惠等产生的折扣优惠金额；
- **折后价**：自2025-05/06月份及之后，折后价=原价-节省计划抵扣原价-优惠金额；之前月份中，折后价=原价-节省计划抵扣原价-优惠金额-抹零金额；
- **代金券抵扣**：通过代金券抵扣的金额；
- **抹零金额**：根据账单抹零金额分摊到对应账单明细的抹零金额；特殊的，同一账单号对应多账单明细时，可能存在部分账单明细的抹零金额为负数；
- **应付金额**：
  - 自2025-05/06月份及之后，结算账单：应付金额=折后价-代金券抵扣-抹零金额=现金支付+信控额度退款抵扣+欠费金额；之前月份中，应付金额=折后价-代金券抵扣；
  - 非结算账单（如试用实例）：无需付款，2023-03及之前的后付费非结算账单该参数为0，2023-04及之后的非结算账单应付金额正常计算但现金支付和欠费金额为0；特殊的，试用实例也不抵扣资源包和代金券；
- **现金支付**：通过账户余额或线上支付渠道支付的金额；特殊地，使用信控赊销的额度在退订/更配时，退款/结转的账单金额体现在现金支付；
- **信控额度退款抵扣**：通过信控额度退款抵减欠费的部分；若信控额度退款抵扣>0，则代表通过信控额度退款抵减的欠费金额；若信控额度退款抵扣<0，则其绝对值代表信控额度退款金额；
- **欠费金额**：已出账未还款的金额；特殊地，信控赊销金额体现在欠费金额中；
- **项目**：资源实例所属的项目信息；取值为结算周期所在小时末实例所属的项目；按天/账期聚合时，项目展示为最新的项目；目前支持展示项目信息的云产品：云数据库 MySQL 版、缓存数据库 Redis 版、弹性块存储、云服务器、公网IP、NAT网关、VPN连接、负载均衡、云企业网、E-MapReduce、云数据库 PostgreSQL 版、文件存储 NAS、文档数据库 MongoDB 版、文件存储 vePFS、应用型负载均衡、DDoS高防、云加密机、消息队列 RocketMQ/Kafka/RabbitMQ 版等；
- **标签**：资源实例的标识；取值为结算周期的最后一小时内资源实例存在过的标签信息；按天/账期聚合时，标签展示为最新的标签；目前支持展示标签信息的云产品：云服务器、弹性块存储、公网IP、容器服务、E-MapReduce、负载均衡、VPN连接、文件存储 NAS、云数据库 MySQL 版、云数据库 PostgreSQL 版、消息队列 Kafka/RabbitMQ 版等；
- **市场价**：抢占式实例的实际计费单价，非抢占式实例取值为空；
- **优惠类型**：账单出账匹配的优惠类型，包括固定单价、超额累进单价、单一折扣、时间满折等；聚合口径下若优惠类型和优惠内容任一字段不完全一致时展示为"-"；
- **优惠内容**：优惠类型对应的具体优惠内容，比如优惠类型为单一折扣时、优惠内容为8折，优惠类型为超额累进单价时，优惠内容为累进阶梯及阶梯单价；
- **有效因子**：按带宽计费时的有效因子；自2023-10账务账期起有效因子有值，2023-10账务账期之前的月份无有效因子时统一展示为0；

### 3. 定制列

可定制要展示的数据字段，定制成功后账单明细页面可展示相应数据。

### 4. 导出

可根据自身需求，导出所需条件下的全量字段账单数据；根据选择的筛选规则，导出对应维度的参数；

- **当前筛选结果**：导出数据与页面展示数据一致；
- **完整表格**：包含筛选条件下的全部账单字段；

## 三、账单总览

### 1. 筛选条件

- **账务账期**：单选，默认为当月；月账单概览支持账务账期范围的选择；
- **账号**：同时查询Payer账号和Owner账号的账单；

### 2. 月账单概览

支持查看所选账务账期范围下，按Payer账号、账务账期聚合后的金额数据、出账状态、还款到期日；

> **注**：还款到期日为月结出账日后，根据您与火山引擎约定的付款周期及账期得出的还款到期日；未约定时默认为次月2号；

### 3. 月趋势

支持按柱状图、折线图查看所选账务账期近半年/一年的消费数据，支持按应付金额、现金支付、折后价分别查看；

### 4. 产品/项目分布

支持查看所选账务账期下，按应付金额、现金支付、折后价分布的前10的产品/项目，包括产品名、消费金额和消费金额占比；

### 5. 汇总

按Payer账号、Seller账号、Owner账号、业务类型、账单大类类型（消费、调账、退款）、账务账期、结算类型汇总的账单数据；

### 6. 产品汇总

按账务账期、Payer账号、Seller账号、Owner账号、产品、业务类型、计费模式、账单大类类型（消费、调账、退款）、结算类型汇总的账单数据；

- **账务账期**：代表账单出账归属账期；
- **Payer账号**：代表此笔账单还款的对象；资源的支付结算账号；财务托管、代销\代售等场景下，Payer账号分别为托管主账号、代销\代售伙伴账号；
- **Seller账号**：代表售卖方；代销代售场景下，seller为服务提供商；
- **Owner账号**：代表真实使用服务的对象，即实际使用资源的账号；财务托管、代销\代售等场景下，Owner账号分别为托管子账号、代销\代售客户账号；
- **产品**：使用的产品/服务的商品名称；
- **业务类型**：业务开展的类型；包含普通业务、财务托管、财务管理、代销业务、代售业务等；
- **计费模式**：使用的产品/服务的计费方式，如包年包月、按量计费、合同计费等；其中合同计费为线下计费，您可联系商务了解计费详情；
- **账单类型**：账单的大类类型，如消费、退款、调账等；
- **结算类型**：标记账单是否需要结算扣款，包含非结算、结算、quota结算；
- **原价**：根据商品目录价计算出来的消费金额；部分线下合同计费账单，不体现优惠，即原价=折后价；
- **节省计划抵扣原价**：通过节省计划抵扣的原价金额；
- **优惠金额**：根据合同折扣、官网活动优惠等产生的折扣优惠金额；
- **折后价**：自2025-05/06月份及之后，折后价=原价-节省计划抵扣原价-优惠金额；之前月份中，折后价=原价-节省计划抵扣原价-优惠金额-抹零金额；
- **代金券抵扣**：通过代金券抵扣的金额；
- **抹零金额**：根据账单抹零金额分摊到对应账单明细的抹零金额；特殊的，同一账单号对应多账单明细时，可能存在部分账单明细的抹零金额为负数；
- **应付金额**：
  - 自2025-05/06月份及之后，结算账单：应付金额=折后价-代金券抵扣-抹零金额=现金支付+信控额度退款抵扣+欠费金额；之前月份中，应付金额=折后价-代金券抵扣；
  - 非结算账单（如试用实例）：无需付款，2023-03及之前的后付费非结算账单该参数为0，2023-04及之后的非结算账单应付金额正常计算但现金支付和欠费金额为0；特殊的，试用实例也不抵扣资源包和代金券；
- **现金支付**：通过账户余额或线上支付渠道支付的金额；特殊地，使用信控赊销的额度在退订/更配时，退款/结转的账单金额体现在现金支付；
- **信控额度退款抵扣**：通过信控额度退款抵减欠费的部分；若信控额度退款抵扣<0，则其绝对值代表信控额度退款金额；若信控额度退款抵扣>0，则代表通过信控额度退款抵减的欠费金额；
- **欠费金额**：已出账未还款的金额；特殊地，信控赊销金额体现在欠费金额中；

### 7. 定制列

可定制要展示的数据字段，定制成功后账单明细页面可展示相应数据。

### 8. 导出

可根据自身需求，选择导出"当前筛选结果"、"完整表格"；

- **当前筛选结果**：导出数据与页面展示数据一致；
- **完整表格**：包含筛选条件下的全部账单字段；

### 9. 导出月账单PDF

- 请于每月第2个自然日12点后查看上月账单数据，若存在计费模式=合同计费的账单时请于次月第4工作日查看上月完整消费；"出账中"状态的账期数据会持续累积；账单总览的汇总数据相比账单数据，延迟1小时更新；
- 月账单PDF仅包含导出账号为Payer账号的结算账单数据；若指定导出条件下无结算账单数据，则不生成PDF文件；
