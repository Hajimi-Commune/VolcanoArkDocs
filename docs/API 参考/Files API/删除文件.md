# 删除文件

`DELETE https://ark.cn-beijing.volces.com/api/v3/files/{file_id}`
根据文件ID删除文件，并将文件从存储空间中移除。
请求参数
路径参数
**id**`string`
待删除的文件id。
响应参数
**id**`string`
被删除的文件id。
**object**`string`
固定为 `file`。
**deleted**`boolean`
文件被删除，取值`true`表明删除成功。

## 示例代码

**Curl**

```bash
curl https://ark.cn-beijing.volces.com/api/v3/files/file-20251014**** \
-X DELETE \
-H "Authorization: Bearer $ARK_API_KEY"

```

**Response:**

```json
{
    "object": "file",
    "id": "file-20251014****",
    "deleted": true
}
```

**Python SDK**

```python sdk
import os
from volcenginesdkarkruntime import Ark

api_key = os.getenv('ARK_API_KEY')

client = Ark(
    base_url='https://ark.cn-beijing.volces.com/api/v3',
    api_key=api_key,
)

if __name__ == "__main__":
    try:
        response = client.files.delete(
            file_id="file-20251114****"
        )
        print(response)
    except Exception as e:
        print(f"failed to delete response: {e}")
```

**Response:**

```json
{
    "object": "file",
    "id": "file-20251014****",
    "deleted": true
}
```

**Go SDK**

```go sdk
package main

import (
    "context"
    "fmt"
    "os"

    "github.com/volcengine/volcengine-go-sdk/service/arkruntime"
)

func main() {
    client := arkruntime.NewClientWithApiKey(os.Getenv("ARK_API_KEY"),arkruntime.WithBaseUrl("https://ark.cn-beijing.volces.com/api/v3"))
    ctx := context.Background()

    fileInfo, err := client.DeleteFile(ctx, "file-20251114****") // update file info
    if err != nil {
        fmt.Printf("delete file error: %v", err)
        return
    }
    fmt.Printf(" delete file: %v", fileInfo)
}

```

**Response:**

```json
{
    "object": "file",
    "id": "file-20251014****",
    "deleted": true
}
```

**Java SDK**

```java sdk
package com.ark.example;

import com.volcengine.ark.runtime.model.files.DeleteFileResponse;
import com.volcengine.ark.runtime.service.ArkService;

public class demo {

    public static void main(String[] args) {
        String apiKey = System.getenv("ARK_API_KEY");
        ArkService service = ArkService.builder().apiKey(apiKey).baseUrl("https://ark.cn-beijing.volces.com/api/v3").build();

        DeleteFileResponse deleteFile = service.deleteFile("file-20251114****");
        System.out.println("Delete File:" + deleteFile);

        service.shutdownExecutor();
    }
}
```

**Response:**

```json
{
    "object": "file",
    "id": "file-20251014****",
    "deleted": true
}
```
