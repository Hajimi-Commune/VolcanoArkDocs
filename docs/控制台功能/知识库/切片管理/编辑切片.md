# 编辑切片
本节将说明如何对切片内容进行编辑。
# 操作步骤
在知识库详情页中的【处理结果】Tab下，可对切片内容进行编辑。
![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/a6ec863c3b314c8b97d07cd9ee09f89d~tplv-goo7wpa0wc-image.image)

点击【编辑】按钮对切片内容进行修改。
:::warning
编辑切片会重新embedding切片，并产生费用。若更新文档，已编辑的切片内容将被更新的文档内容覆盖。
:::
