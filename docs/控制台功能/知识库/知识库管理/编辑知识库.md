# 编辑知识库
本节将说明如何编辑知识库
:::tip
知识库仅支持编辑描述、标签和 CPU 配额，其他配置均不可编辑。
:::
# 操作步骤

1. 在[知识库列表页面](https://console.volcengine.com/ark/region:ark+cn-beijing/knowledge/collection/list)，hover 需要编辑知识库卡片右上角的...，点击「编辑」，进入**编辑知识库**页面。或进入知识库详情页中，点击右上角的「编辑」，同样可以进入**编辑知识库**页面。

![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/185ba164b2c74222bd4821810cbbd692~tplv-goo7wpa0wc-image.image =1830x)

2. 在**编辑知识库**页面可修改知识库描述、标签和CPU配额。编辑完成后单击右侧「更新知识库」即可。

![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/84d9a9a06d224fb08c7a4c515c4ee128~tplv-goo7wpa0wc-image.image =2411x)
