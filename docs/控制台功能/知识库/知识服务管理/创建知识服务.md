# 创建知识服务
当在知识库控制台对问答效果进行调优后，通过“知识服务”可以记录这一套最佳参数配置组合，并通过 API 便捷集成到自己的业务中
# 操作步骤
1、在知识库控制台上调试参数，获得预期效果
![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/a36a91590572474eb07a3aa386432c51~tplv-goo7wpa0wc-image.image)
2、点击【创建服务调用】配置知识服务基本信息，并至少关联一个可用的 API key
![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/bb19c93221ff4309b2347245687df520~tplv-goo7wpa0wc-image.image)
3、点击【确认创建】即可完成知识服务的创建，平台提供多种常见开发语言的调用样例，可一键复制集成
![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/638d473e0edf47aeba0ed0e2b18e51bc~tplv-goo7wpa0wc-image.image)
此时，通过 API 调用即可得到与控制台完全一致的效果
