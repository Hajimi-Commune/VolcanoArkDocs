# 语音模型
# 进入体验中心
可从以下位置进入体验中心：

* 在控制台左侧导航栏中，点击 [体验中心](https://console.volcengine.com/ark/region:ark+cn-beijing/experience/chat)。
* 在 **模型广场** 中打开模型详情页，点击 **立即体验** 进入体验中心。

# 试用模型
语音模型包括同声传译、实时语音、声音复刻、语音合成、语音识别模型。
下文以语音合成模型为例，介绍试用方法。

1. 在体验中心界面左上方展开模型列表，在 **语音** 标签页中选择语音合成模型。
   ![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/9449ef11fa414d38b727949108b88363~tplv-goo7wpa0wc-image.image =1394x)
2. 在右侧选择音色并配置情感等参数。
3. 输入文本或选择预设文本。
4. 点击 **合成试听**，待语音合成完毕后，即可开始试听。

# 其他说明补充
语音合成的免费合成额度为 20,000 字符，您可在页面点击 **开通语音模型** 开通。
目前该体验只对企业客户开放，如需测试或接入须先进行企业认证。
