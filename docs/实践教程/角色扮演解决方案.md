# 角色扮演解决方案
**一、大模型角色扮演，也叫捏角色，是基于大语言模型出色的语音理解能力和自然的交互方式通过角色扮演为用户提供更逼真和个性化的体验。目前被广泛应用在以下场景：**

* 社交/AI陪聊/ChatBot
* 强风格化的客服/老师/助手等toC的角色
* 体验更好的智能NPC
* 强风格化的IP角色（企业形象、明星、热门人物等）

通过大模型塑造角色会面临一些挑战，比如如何轻量的创作人设，如何保持人设的一致性，如何能让角色的对话能力和剧情演绎能力不断增强并保持长期记忆。基于以上的挑战，我们推出了一套适配于角色扮演的解决方案。
