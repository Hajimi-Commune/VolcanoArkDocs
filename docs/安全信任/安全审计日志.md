# 安全审计日志
基于[火山方舟互信计算架构](https://www.volcengine.com/docs/82379/1099355)，字节终端安全部门在互信服务安全沙箱内置了监控[Linux System Call](https://man7.org/linux/man-pages/man2/syscalls.2.html)组件提供审计数据，展示对用户模型服务的连接和异常进程监控，保证用户的模型不被意料之外的操作获取

- 依靠监控[Linux System Call:Connect](https://man7.org/linux/man-pages/man2/connect.2.html)信号，连接清单展示容器内主动发起连接（包含向内和向外连接）的请求情况，暂无详细连接数据量
- 依靠监控[Linux System Call:Execve](https://man7.org/linux/man-pages/man2/execve.2.html)信号，异常进程清单展示和SSH/WebShell相关的登录信息

# 审计日志模块
平台提供审计日志模块，用户可在此查看近30天内使用过的互信服务的异常日志。在无异常情况下，页面样式如下
![alt](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_e421bc7dc5ae60f3a180b2e4aec33a68.png)
若平台检测到异常，会展示异常实例及异常实例下的具体异常连接和异常进程清单，如有异常，您可点击「联系火山方舟」，方舟平台将协助您排查异常，保障您的模型安全
![alt](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_af030a553a25aea4b92f002bc0ebd7a7.png)
# 支持模型维度查看安全审计日志
对于模型Vendor，平台提供模型维度的安全审计日志。可在模型详情页面，切换至「安全审计日志」，选择当前模型部署的**互信服务**，查看对应安全审计日志
![alt](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_8d5b9e40194de73284b189fd5e409a26.png)
选择服务后，可查看服务下每个容器实例的连接清单及异常进程清单。
![alt](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_b2fb7b946897e037ab18dca953035f80.png)
# 在模型服务详情页查看安全审计日志
用户可在模型服务详情页切换至「安全审计日志」，查看该模型服务的安全审计日志
