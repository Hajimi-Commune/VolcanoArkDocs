# PromptPilot 管理
## 管理任务
支持查看、管理Prompt调优或Solution探索任务。每个任务对应一张任务卡片。
[火山方舟版本入口](https://console.volcengine.com/ark/region:ark+cn-beijing/autope/tasklist)
[独立站版本入口](https://promptpilot.volcengine.com/tasklist)
## 知识库
支持创建、查看、管理您的知识库。创建知识库后，可在 **Prompt生成** 模块使用，全方面提升Prompt优化、模型回答以及数据构造效果。 
火山方舟版本入口：[登录 Prompt 管理页面](https://console.volcengine.com/ark/region:ark+cn-beijing/autope/tasklist)，单击 **知识库管理**。
独立站版本入口：[登录任务管理页面](https://promptpilot.volcengine.com/tasklist)，单击左下角 **知识库** 图标。
## 精调历史（仅火山方舟版本）
查看精调历史及其提升效果。在使用「评分模式」的「文本理解/单轮对话」任务中，可在「智能优化」流程中开启「免费智能精调」，当生成Prompt优化结果后，平台将智能判断是否自动进入精调继续提升优化效果，若有精调结果将更新报告，当前精调方法为 SFT精调，用理想回答作为训练目标。
[火山方舟版本入口](https://console.volcengine.com/ark/region:ark+cn-beijing/autope/tasklist)
## API Key
支持创建、查看、管理您的 API Key。API Key 是用户通过 API/SDK 调用PromptPilot服务的重要凭证。API Key 长期有效，请不要将密钥信息共享至公开环境，妥善保管并定期轮换密钥，避免因未经授权的使用造成安全风险或资金损失。
火山方舟版本入口：[登录 Prompt 管理页面](https://console.volcengine.com/ark/region:ark+cn-beijing/autope/tasklist)，单击 **Get API Key**。
独立站版本入口：[登录任务管理页面](https://promptpilot.volcengine.com/tasklist)，单击左下角 **API Key** 图标。
