# 协作奖励计划规则（第二期）和数据授权协议

**活动简介**
**活动入口**
您可在[火山方舟控制台-开通管理页](https://console.volcengine.com/ark/region:ark+cn-beijing/openManagement)，点击 **立即参与** 进入产品活动页面。 如您未完成实名认证，请在实名认证后参与。
如您尚未开通模型服务，请开通模型服务后参与；如您开启了[安心体验模式](https://www.volcengine.com/docs/82379/1465347)，请关闭安心体验模式后参与。
**活动参与范围**
完成企业认证的用户可参与本活动，享受企业权益。
完成个人实名认证的用户可参与本活动并享受个人权益，可在活动页面完成企业认证，并手动确认升级，享受企业权益。
**活动规则说明**
首次授权模型及推理接入点参与本计划，可获得一个该模型的冷启动免费资源包，该资源包可用于抵扣按tokens后付费用量，奖励额度以【**产品活动介绍页**】公示为准。
调用已获授权的推理接入点，每日将采集每个模型不超过500万tokens的推理数据，并于次日发放有效期为30天的等量免费资源包；未获授权的接入点数据将不予采集，该部分使用量次日也不会返还资源包。
个人实名认证用户可享有个人权益，完成企业认证升级后可享有企业权益，采集量及免费资源包上限将提升至企业权益对应水平，具体详情请参阅活动页面说明。
本计划所发放的免费资源包，可用于抵扣在线推理过程中推理输入、推理输出以及推理缓存命中的按token后付费计费用量。目前，暂不支持抵扣批量推理、按模型单元付费、TPM保障包以及推理缓存存储等计费项的用量。若后续对上述抵扣规则进行调整，将在相关活动页面或文档中予以说明，请予以关注。
目前平台仅针对部分模型和调用方式进行数据采集，通过不支持的方式调用模型时，数据不会被采集，次日也不会发放该部分对应的资源包奖励。当前支持的调用方式：
Chat API
Responses API
Messages API
Image Generation API
Content Generation API （仅在线推理）
支持采集的模型以【**产品活动介绍页**】公示为准，若后续有调整，将在相关活动页面或文档中予以说明，请予以关注。
近期本计划参与人数较多，资源包下发可能存在延迟，请耐心等待。
**数据安全保障**
隐私无忧：您的授权数据会被严格加密和匿名化处理；
无授权不采集：当您从未授权、撤回授权，或者本计划终止时，您的数据不会被采集；
授权自主：您可以随时在数据协作奖励计划活动页面撤回授权。
火山**方舟协作奖励计划活动规则（第二期）**
火山引擎邀请客户（“您”）参与协作奖励计划（“本计划”），本计划旨在与您一起共建更优质的模型服务。
**协作奖励计划权益说明**
本计划合作期（**原合作期为2024年12月1日至2025年5月31日，合作期延长至2026年3月31日**）内，在您订购/使用火山方舟指定模型时（以【**产品活动介绍页**】公示为准）如选择参与本计划，火山方舟将为您提供一份冷启动资源包，同时为您提供每日奖励包（统称“方舟奖励资源包”或“奖励包”）。
（1）奖励包的免费资源额度和种类等，以您开通本计划时火山方舟指定模型在【**产品活动介绍页**】公示的奖励信息为准；
（2）冷启动资源包的奖励机制为：在您首次成功授权某模型的推理接入点后，您将立即获得该模型免费额度资源包奖励，具体奖励额度以活动页面显示为准。
（3）每日奖励包的奖励机制为：在您授权火山方舟指定模型的推理接入点参与协作奖励计划后，将在每日额度内自动按顺序采集授权模型推理接入点对应的调用数据，次日根据每个模型的前一日采集量返还等量的每日奖励包。
（4）奖励包仅可用于抵扣按 Token 付费的后付费推理费用；
（5）奖励包将在发放到您账号后的 30 个自然日内有效，到期后清零；
您可选择同时使用参与本计划的任意一款或几款模型服务，每一款模型将按模型推理接入点申请获取您的数据授权并据此为您提供奖励包。
本计划仅支持同一用户的唯一个账号参与，即如您同一主体下有多个账户或多个账号，这些账户或账号不可同时申请参与本计划。
**您在合作期内调用火山方舟指定模型服务期间的调用数据，即因调用模型而输入和模型生成的内容，包括但不限于所有输入的文本、声音、图形、图片、图像、视频、软件或任何其他类似信息（以下称“协作奖励数据”）将授权提供给火山引擎用于火山引擎及其关联公司的模型和算法优化。**
如您已参与协作奖励计划第一期活动，即 5亿资源奖励计划，您可以同时参与本计划，您的权益将不受影响。您的具体权益详见[协作奖励计划规则和数据授权协议](https://www.volcengine.com/docs/82379/1391984)。

**您理解并确认，火山引擎可永久使用您在合作期内已经授权给火山引擎的协作奖励数据。若您已参与本计划，但在参与过程中希望撤回授权，您可以在当前页面点击“撤回授权”按钮，那么您后续的调用数据将不再被提供给火山引擎，且本奖励计划将自行终止，您未使用完毕的奖励资源包仍然可在有效期内继续使用。如您在本计划合作期（包括火山引擎根据运营情况，调整后的活动有效期）内未主动撤回授权，那么在合作期内，您的授权将持续有效。**
其他数据授权相关约定，详见下方《火山引擎数据授权使用协议》。
