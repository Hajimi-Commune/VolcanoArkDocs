# 旧版账单管理

## 账单出具规则

- 2021年12月22日之后的账单数据已包含预付费（未包含更配、退订订单）和后付费的全部消费数据，2022年1月20日之后包含预付费更配、退订订单数据；
- 系统按月结算的产品，当月账单最终在次月第3个工作日生成；
- 自2021年8月1日起，账单明细费用支持到6位小数，汇总为账单后，抹掉3位及之后的小数，实际从账户扣费时按2位小数进行扣费（即扣到分）。在此之前出具的账单，账单明细以不足0.01元按0.01元取值为准。

## 账单扣款规则

- 出具账单后，按照出账先后顺序进行扣款；
- 优先抵扣代金券，超出部分使用账户余额抵扣；（代金券抵扣规则参考[代金券管理说明文档](../代金券管理/代金券管理.md)）
- 用户充值后，默认按照出账先后顺序，进行未结清账单还款；

## 一、账单明细

### 筛选条件

- **账期**：单选，默认为当月；
- **账单号**：按账单号精确搜索；
- **实例ID/实例名称**：支持按实例ID/实例名称精确搜索；

### 表格参数

- **账期**：使用产品/服务对应的自然月；
- **产品**：云产品的名称；
- **消费时间**：后付费计费开始时间和结束时间；
- **账单号**：预付费类型下订购的订单号/用户后付费类型下消费的账单号；
- **实例ID/名称**：账单对应的实例ID及实例名称；
- **计费单元**：实例/计费项的计费单元信息；
- **地域**：实例/计费项的地域信息；
- **可用区**：实例/计费项的可用区信息；
- **影响因子**：实例/计费项的价格影响因子信息；
- **扩展字段**：实例/计费项的备注信息；
- **单位**：用量的单位；
- **用量**：产品/服务的使用量；
- **资源包抵扣量**：产品/服务使用资源包抵扣的量；
- **账单原价**：根据商品目录价计算出来的消费金额；部分线下合同计费账单，不体现优惠，即原价=折后价；
- **优惠金额**：根据合同折扣、官网活动优惠等产生的折扣优惠金额；
- **折后价**：应付金额=原价-优惠金额-抹零金额；
- **订单号**：展示预付费订单号；

### 定制列

可定制要展示的数据字段，定制成功后账单明细页面可展示相应数据。

### 导出

包含页面所选账期下的全部账单字段。

## 二、账单

### 筛选条件

- **账期**：单选，默认为当月；

### 表格参数

- **账期**：使用云产品对应的自然月；
- **产品**：云产品的名称；
- **消费时间**：预付费支付时间、后付费计费开始时间和结束时间；
- **账单号**：用户后付费类型下消费的账单号；
- **账单原价**：根据商品目录价计算出来的消费金额；部分线下合同计费账单，不体现优惠，即原价=折后价；
- **优惠金额**：根据合同折扣、官网活动优惠等产生的折扣优惠金额；
- **抹零金额**：账单明细存在多小数位，汇总为账单时，产生的小数点后3位及以上不扣费直接抹除的金额；
- **折后价**：应付金额=原价-优惠金额-抹零金额；
- **代金券抵扣**：通过代金券抵扣的金额；
- **应付金额**：结算账单：应付金额=折后价-代金券抵扣=实付金额+待付金额；非结算账单（如试用实例）：无需付款，该参数为0；
- **实付金额**：通过账户余额或线上支付渠道支付的金额；
- **待还款金额**：已出账未还款的金额；
- **还款状态**：包含待还款、部分还款、已还款等；
- **订单号**：展示预付费订单号；

### 定制列

可定制要展示的数据字段，定制成功后账单明细页面可展示相应数据。

### 导出

包含页面所选账期下的全部账单字段。
