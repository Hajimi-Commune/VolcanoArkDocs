# 豆包助手计费

豆包助手 API 通过火山方舟平台提供「豆包同款」AI 能力，支持 **日常沟通、深度沟通、联网搜索、边想边搜** 四大核心功能，按次计费，即开即用，无需复杂调试。
**计费概述**
豆包助手采用后付费模式，按次计费。不同功能的收费标准不同，详情请参见[计费项](/docs/82379/1998171#73f316ae)。
调用豆包助手时，其涉及的底层模型调用不单独计费，但您需要确保相应的模型服务已开通且可以正常使用。
计费项
*
开通服务
使用豆包助手功能前，您可以在火山方舟平台 [开通管理页面](https://console.volcengine.com/ark/region:ark+cn-beijing/openManagement?LLM=%7B%7D&OpenModelVisible=false&tab=DoubaoAssistant) 一键开通豆包助手。
说明
豆包助手功能暂未全量开放，请联系对应的销售人员或提交工单进行申请开通。
开通后，您可以通过 API 调用豆包助手，请参见[豆包助手](/docs/82379/1978533)。
欠费说明
后付费账单按小时结算，出具账单后实时结算扣款。如果您账户中的可用额度（含账户余额和代金券）小于待结算的账单，会被认为欠费。
豆包助手依赖火山方舟模型服务，因此其欠费规则与火山方舟模型服务保持一致。火山方舟于 2025年7月16日起，**出具账单后欠费1分钟即关停**。
为避免服务因欠费中断，请您密切关注账户的支付状态。如需保障稳定的线上服务，可开启 [延期免停权益](https://www.volcengine.com/docs/6269/202201)。
*
如果确定不再使用服务，请在开通管理页面[关闭服务](/docs/82379/1998171#a1f3a89e)。
请及时补缴所有欠费账单以避免服务中断带来的影响。请前往 [火山引擎费用中心](https://console.volcengine.com/finance/fund/recharge) 进行充值，详细操作参见 [充值操作指引](https://www.volcengine.com/docs/6269/100434)。
关闭服务
如果您决定关闭服务，可在火山方舟 [开通管理页面](https://console.volcengine.com/ark/region:ark+cn-beijing/openManagement?LLM=%7B%7D&OpenModelVisible=false&tab=DoubaoAssistant) 进行相应操作。
注意
关闭服务将导致豆包助手 API 无法调用。
请在操作前确认您的业务不再依赖此服务，以免造成业务中断。
