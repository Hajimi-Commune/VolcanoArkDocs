# 共享数据集
通过共享数据集功能，您可将您的数据集及所包含的所有数据集版本共享给其他用户使用
<br>

在获取 资源共享功能 的IAM权限后，您可至 [资源管理](https://console.volcengine.com/resourcemanager/region:resource-manager+cn-beijing/resource_share/owned) 共享您的数据集和接受其他用户的共享，流程如下。

## 账号开通资源共享权限

没有资源共享权限的用户，无法查看接收到的共享资源，需要先配置“资源共享”的IAM权限策略。
（1）由账号负责人（主账号或管理员）通过页面右上方头像点击「访问控制」，或点击跳转到控制台的[策略管理](https://console.volcengine.com/iam/policymanage?scope=All&service=resource_share)页。
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_26f79611f2da0a658200f04c9a67810c.png)
（2）在「策略管理」中，搜索“资源共享”产品，选择[ResourceShareFullAccess](https://console.volcengine.com/iam/policymanage/System/ResourceShareFullAccess)策略，点击“管理”操作。
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_2191dc9825e2797172d1b98a437a1c31.png)
（3）在策略详情中，点击“添加授权”，选择需要授予权限的用户、用户组或角色，确认后完成权限授权。
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_17a9cea62d743e85a10abfa6c95594c4.png)
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_0f09d2de6237eb9693489543a676ada4.png)
（4）修改共享权限：用户可以在策略详情中修改IAM身份的策略权限范围，或移除权限。
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_c6ee013ba00671ce153e5ba251956359.png)
<br>

## 发起共享

![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_605788d0e9ed100c53573ce6f1f158f9.png)
（1）获取资源共享权限后，进入「资源共享」控制台
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_f8e1a18c7ca6b668d9cf2f5a1e66600f.png)
（2）点击「创建共享单元」进入创建流程。
选择所需要共享的数据集资源，资源ID和数据集ID相同，可在「我的数据集」列表页查询到所需要共享的数据集ID。
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_3da94175670bd6115db54767c1a9475b.png)
（3）选择预置策略，填写被共享账号的ID，发起共享。
更多说明请参考：[资源共享-由我共享管理](https://www.volcengine.com/docs/6649/1220179)
<br>

## 接受共享邀请

拥有共享权限的用户可以进入「资源共享」控制台，接受共享邀请。
（1）点击页面右上方头像，选择「资源管理」进入「资源共享」控制台，在左侧导航栏选择「[共享给我](https://console.volcengine.com/resourcemanager/region:resource-manager+cn-beijing/resource_share/shared)」，可查看收到的共享。
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_2a2c501de949b004dd56e2065d1451d9.png)
（2）收到的共享邀请会在左侧菜单栏显示，用户进入「共享给我」后可查看收到的全部共享单元。
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_8dc99b3b49159e8f91d5cfa97b2bc618.png)
（3）点击“接受”共享邀请后，就可以访问共享资源。您可以前往「火山方舟」[数据集-共享数据集](https://console.volcengine.com/ark-stg/region:ark-stg+cn-beijing/dataset?Common=%7B%7D&Shared=%7B%7D&projectName=wz&tab=%22Shared%22) 查看并使用接收的数据集。
<br>

更多说明请参考：[资源共享-共享给我管理](https://www.volcengine.com/docs/6649/1220178)

## 使用共享数据集

### 精调

![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_94a41e6aee26ea6b39a87820f57640a2.png)
（1）在模型精调时，选择好训练类型和模型后进行数据配置。使用「从数据集选择」方式，添加训练集和验证集。
![](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_3b8cbd9a3e04a72a50a4a65bd6f764c1.png)
（2）点击「选择数据集」选择合适的共享数据集和数据集版本，添加为精调的训练集或验证集。每个精调任务可选择不超过10个数据集进行训练或验证。
（3）添加并通过格式校验后，提交精调任务进行精调。
