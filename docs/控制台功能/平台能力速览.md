# 平台能力速览
# 前提条件

1. 注册账号：在使用火山引擎的服务之前需要用户注册火山引擎账号并完成相关认证，详见[账号注册流程](https://www.volcengine.com/docs/6261/64925)。
2. 实名认证：在火山引擎上购买和使用云资源的过程中，为遵守相关法律法规，也为保障您的合法权益，需要先进行实名认证，您可以按需进行个人实名认证或者企业实名认证，详见[实名认证基本介绍](https://www.volcengine.com/docs/6261/64935)。
3. 创建子账号(可选)：如果您的账号为多人使用，为更好地进行权限管理，建议使用主账号前往[【访问控制】-【用户】](https://console.volcengine.com/iam/identitymanage/user)创建子账号并通过预置策略为其分配适当的权限。

# 模型与推理体验
## 模型广场
完成账号与权限设置后，进入火山方舟控制台（[访问地址](https://console.volcengine.com/ark)），在**模型广场**查看目前已发布的大模型，点击某个模型卡片可查看该模型的详细信息。更多详情可见[模型广场介绍](/docs/82379/1099503)。
![Image](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_2d86f4d69e1c4ad5c7e3f74799d4b12a.png)

**模型仓库**展示所有精调产出的模型，您可以在此处管理自己的模型，查看模型的基本信息、接入点信息，发起评测、进行推理体验等，更多详情可见[管理自定义模型](/docs/82379/1217587)。
![Image](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_70ac8a28155364d5d5ec422aec85422e.png)

![Image](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_af4c062218634a0ee89fcf02b9d368ec.png)

# 模型精调与评测
火山方舟允许用户上传自己的数据进行模型精调，只需简单的几个步骤即可产出更加贴合客户业务场景的模型。
## 准备数据
在创建精调任务之前需要完成数据的准备，目前支持 JSONL 格式的数据集，详细说明请参考[模型精调数据集格式说明](/docs/82379/1099461)。目前支持两种数据上传的方式：

1. 从本地上传数据：适用于小规模的数据集。
2. 从 TOS 中导入数据：需先将数据上传至 TOS，具体操作可见[TOS控制台快速入门](https://www.volcengine.com/docs/6349/74830)，然后在创建精调任务时选择对应的 TOS 目录。当数据集规模较大时建议使用这种方式，避免在上传过程中因网络问题导致传输中断。

## 创建模型精调任务

1. 数据准备好后，可以去创建模型精调任务，平台提供两种入口，您可以按照自己的需求进行选择。
   1. 入口一：进入**模型精调页面**，点击**创建精调任务**
      ![Image](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_9e644456535fe9120b521ba8ea47e5b2.png)
   2. 入口二：通过模型广场的模型详情页中，**模型精调**发起精调任务
      ![Image](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_4df14aba703acc3b13c4290e2015fe4b.png)
2. 进入**创建精调任务**页面，按实际情况进行任务名称、描述、基础模型、数据集等信息的选择或填写。
3. 数据填写完毕后，点击**提交精调任务**，即可完成任务创建。 更多操作，可见[创建模型精调任务](/docs/82379/1099459)。

![Image](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_f91e1fb50a51403b1c76a8fb9caf9951.png)
## 查看并管理模型精调任务

1. 进入**模型精调**列表页，可查看已提交的所有模型精调任务。
2. 点击某个任务名称，即可进入对应的详情页，可查看基本信息、效果指标、日志、时间线、模型产出信息。

更多操作，可见[查看并管理模型精调任务](/docs/82379/1099460)。
![Image](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_71e189a376efe38db1cd78d8d9633793.png)
## 创建模型评测任务

1. 模型训练完成后，可以去创建模型评测任务，平台提供三种入口，您可以按照自己的需求进行选择。
   1. 入口一：进入**模型评测**页面，点击**创建评测任务**
      ![Image](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_98b976c721c5b03c5b41f5f51db6e6b6.png)
   2. 入口二：进入精调任务详情页，切换**模型产出** tab，选择模型后发起评测
      ![Image](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_c08b8ba4a3577779c7bb28c89753926d.png)
   3. 入口三：进入**模型仓库**页面，选择模型后发起评测
      ![Image](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_4ee47207899be21d3ead8ca72b7421ae.png)
2. 进入**创建评测任务**页面，按实际情况进行任务名称、描述、评测模型、数据集等信息的选择或填写。
3. 数据填写完毕后，点击**提交评测任务**，即可完成任务创建。 更多操作，可见[创建模型评测任务](/docs/82379/1150782)。

![Image](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_f637663f50fc147e3c62a95caf653609.png)
## 查看并管理模型评测任务

1. 进入**模型评测**列表页，可查看已提交的所有模型评测任务。
2. 点击某个任务名称，即可进入对应的详情页，可查看基本信息、评测报告、日志。

更多操作，可见[创建模型评测任务](/docs/82379/1150782)。
![Image](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_2ca05d6d90d88d5c1dc37ed3096e2ad6.png)

## IAM 权限管理实践
当您账号涉及到多人使用，或者为组织所有，您可以使用 IAM 来实现不同账号的权限管理。
访问控制 IAM (Identity and Access Management) 是一套权限管理系统，是火山引擎提供的**免费服务**之一。您可以为火山引擎账号（主账号）创建IAM用户（子账号），并根据不同用途授予子账号相应的访问权限，以达到不同企业员工之间的权限隔离。
> 例如，您组织中有负责运营的人员，您不希望他们发生如删除模型等高危操作，您可以配置子账户的权限为ArkReadOnlyAccess策略，允许其仅可查看方舟内的全部资源，但不允许对资源进行操作，从而提升资源的安全性。

更多关于权限管理的详细实践，如项目维度权限管理、用户组权限管理，请参见[使用IAM进行访问控制教程](/docs/82379/1263493)。
