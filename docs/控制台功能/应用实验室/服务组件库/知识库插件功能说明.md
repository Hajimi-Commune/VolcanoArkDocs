# 知识库插件功能说明
“知识库”插件提供了知识管理的能力，将本地上传、 TOS 中存储或指定链接的多个文档导入到知识库中，并对文档执行解析、切片、向量化、构建索引等处理流程，处理完成后即可进行知识检索。

- **插件名称** | **组件说明** | **配额限制** | **计费说明**
- 知识库 | * 支持本地上传、从 TOS 中导入、公开下载链接、从飞书文档导入。
- * 支持多种文档格式，详情请参见[支持文档格式说明](/docs/82379/1261890#be53bf7d)：
- * 非结构化文档：pdf、doc、docx、pptx、markdown、txt、faq.xlsx
- * 结构化文档：jsonl、csv和xlsx。 | 参见[知识库配额说明](/docs/82379/1343907) | 参见[知识库计费](/docs/82379/1263336)。
- 从知识库搜索到的内容将被作为输入的一部分送入大模型，故此消费更多tokens。

## **功能特点**

* **文档处理全面**：能解析多种格式文档，含图片、表格的文档可自动提取文本和表格，开启【图片OCR】还能识别图片文字。
* **检索模式多样**：多种文本向量化模型可选，部分支持混合检索。检索时可调整参数，如denseweight控制语义检索比重。
* **多轮问答智能**：【知识问答】支持多轮问答，可基于历史对话改写问题，有多种模型参数可配置，还能查看召回详情和tokens消耗。

## **注意事项**

* **地区限制**：目前仅支持华北地区。
* **数量限制**：一个用户最多创建200个知识库，每个知识库最多传入50万篇文档，详情请参见[知识库配额说明](/docs/82379/1343907)。
* **计费规则**：按量后付费。从上传文档开始计费，文档删除不影响计费，删除知识库才停止。详情请参见[知识库计费](/docs/82379/1263336)。

## 基本用法

* 控制台体验：用户可通过零代码应用创建的可视化界面进行接入与使用，详情请查看 [零代码应用操作指南](https://www.volcengine.com/docs/82379/1267885)。
* API 调用：支持通过 BotChatCompletions 接口调用已添加知识库插件的零代码应用，详情请查看 [Bot API 文档](https://www.volcengine.com/docs/82379/1285207) 与 [Bot SDK 文档](https://www.volcengine.com/docs/82379/1330201)。
