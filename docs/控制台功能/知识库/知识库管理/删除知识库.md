# 删除知识库
# 操作步骤

1. 在知识库列表页面，hover 需要删除知识库卡片右上角的 【···】，点击【删除】，弹窗提示确认是否删除

![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/5f2a6172c97f49029634e7cb48b5e0bf~tplv-goo7wpa0wc-image.image)

2. 或进入知识库详情页中，点击右上角的【删除】，同样可以触发删除弹窗

![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/9c53e0b8c21b4f5c9fdf6ce0dfe395b8~tplv-goo7wpa0wc-image.image)
