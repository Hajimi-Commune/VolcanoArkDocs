# 合同管理概览

火山引擎控制台合同管理为客户提供订单合同、产品报价合同、信控合同、代付合同等四种合同类型的申请、签约与查询相关功能。
客户通过控制台申请或签约的合同均会加盖火山引擎电子签章，电子版合同与纸质合同具有同等法律效力。

## 合同类型

- **[订单合同](订单合同.md)**：您可基于您在火山引擎的预付费订单或后付费账单申请合同，满足请款或报销等用途的场景。您可通过页面自助申请订单合同，申请后会加盖火山引擎电子章。

- **[产品报价合同](产品报价合同.md)**：该合同包含火山引擎商务经理为您申请的商务价格优惠。合同完成审批后会推送到您的控制台，您可通过控制台页面决定是否签署该合同，同意签约后会加盖火山引擎电子签章。

- **[信控合同](信控合同说明.md)**：该合同包含火山引擎商务经理为您申请的授信信息。合同完成审批后会推送到您的控制台，您可通过控制台页面决定是否签署该合同，同意签约后会加盖火山引擎电子签章。

- **[代付合同](代付合同.md)**：当您需要使用与您实名认证主体不一致的银行账户进行对公汇款时，需要先签署三方代付合同才可以直接向您的火山引擎专属账户进行汇款充值。
