# 用量统计
在 [用量统计](https://console.volcengine.com/ark/region:ark+cn-beijing/usageTracking) 页中，您可查看主账号下对应项目的调用量统计，也可按照在线推理、体验中心、批量推理等维度分别统计这些维度的用量。
![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/7ba83bbb9b6d4c548f8fa6993803f2fe~tplv-goo7wpa0wc-image.image =1284x)
其中：

* 在线推理支持按照天/小时粒度查看
   * 当前项目使用量：调用总tokens数、输入总tokens数、输出总tokens 数，并可按照预置推理接入点/自定义推理接入点进行拆分。
   * 指定接入点的使用量：全部、模型输入、模型输出用量。
* 批量推理支持按照推理任务、推理接入点维度查看用量。
   * 全部、推理任务、推理接入点。
   * 接入点详情：全部、模型输入、模型输出 维度。
