# 套餐概览
方舟 Coding Plan 是为广大开发者量身打造的 AI Coding 场景订阅服务，现已全面支持 DeepSeek-V3.2。该服务此前已支持 Doubao-Seed-Code，如今新增 DeepSeek 的强大编程能力，兼容 Claude Code、Cursor、Cline、Codex CLI 等主流 AI 编码工具，为开发者提供畅快、智能、稳定的编码体验，提升代码编写效率与质量。
:::tip
方舟 Coding Plan聚焦**个人开发场景**，助力开发者完成个人项目、学习实践、工具搭建等编码任务。企业级开发需求，请通过**火山方舟模型 API** 调用支持的模型服务。
:::
## 方舟 Coding Plan 活动

* 参与活动：单击[方舟 Coding Plan活动](https://www.volcengine.com/activity/codingplan)，可直接订阅套餐。
* 快速开始：套餐订阅完成后，可以参考[快速开始](/docs/82379/1928261)，开启 AI Coding 全新体验。
* 活动细则：[套餐概览](/docs/82379/1925114)。

## 套餐详情
### 套餐用量说明

- 套餐 | 适用场景 | 用量限制
- Lite套餐 | 中等强度的开发任务，适合大多数开发者。 | * 每5小时：最多约 1,200 次请求。
- * 每周：最多约 9,000 次请求。
- * 每订阅月：最多约 18,000 次请求。
- Pro套餐 | 复杂项目开发，适合高强度工作的开发者。 | > Lite套餐的5倍用量。
- * 每5小时：最多约 6,000 次请求。
- * 每周：最多约 45,000 次请求。
- * 每订阅月：最多约 90,000 次请求。

:::warning
* 实际用量会因开发任务复杂度、代码库大小及 Coding 工具的使用方式等因素有所不同。
:::
### 套餐额度刷新规则
套餐额度在时间周期内耗尽后，需要等待下一个周期自动恢复额度，不会消耗其他资源包或账户余额。

* 每5小时限额：根据实际请求发生时间，滑动5小时窗口动态刷新。
* 周限额：每周一00:00:00重置周限额。
* 月限额：每订阅月第1日00:00:00重置月限额。

### 套餐额度使用限制

* 同一订阅套餐可在所有支持的工具中使用，额度共享。
* 套餐额度仅在支持的编程工具中生效，不可用于API调用。在非 AI 编程产品中使用方舟 Coding Plan 权益对应的 Base URL 和 API Key 有可能被识别为滥用/违规，会导致订阅停用或账号封禁。

## 接入编程工具
:::warning
* 务必使用指定模型及 Base URL。
   * 其他模型无法在 Coding Plan 中使用。
   * 如未使用指定的 Base URL，将无法使用 Coding Plan 额度，并可能产生额外 API 请求的费用。
:::
使用 Coding Plan 套餐接入编程工具时，需要关注：

* 支持的模型：`doubao-seed-code-preview-latest`、`ark-code-latest`(对应模型 DeepSeek V3.2)
* Base URL：`https://ark.cn-beijing.volces.com/api/coding`
* 兼容 OpenAI API 工具Base URL：`https://ark.cn-beijing.volces.com/api/coding/v3`

Coding Plan 套餐适用于 Claude Code、Cursor、Cline、Codex CLI 等编程工具，可参考以下文档接入并使用。

* Claude Code：[快速开始](/docs/82379/1928261)
* 其他编程工具：[接入AI编程工具](/docs/82379/1928262)

在编程工具中使用 Coding Plan 套餐时可能会遇到报错，请查看对应的报错场景并定位问题，具体可参考[错误码](/docs/82379/1299023)。
