# 查看知识库详情
# 操作步骤
在知识库列表中，点击知识库卡片，即可进入知识库详情页面。

1. 在详情页中【原始文档】tab 下，可查看知识库已上传文档列表，及其处理状态

![图片](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/baf4ff3425f645e7b247ea6163b825e8~tplv-goo7wpa0wc-image.image)

2. 在详情页【处理结果】tab 下，可查看处理后的文档切片内容
![图片](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/3d13c3e41f3e4d038a933988a0885f33~tplv-goo7wpa0wc-image.image)

3. 点击右上角的【知识库信息】，可查看知识库配置

![图片](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/97af72e567204bf7b03c4b6273c85182~tplv-goo7wpa0wc-image.image)
