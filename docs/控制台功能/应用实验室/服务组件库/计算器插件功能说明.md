# 计算器插件功能说明
## 功能简介
计算器插件，通过增强Doubao-pro系列模型的数学能力，为用户提供准确高效的计算服务。该插件可集成至应用并广泛应用于学习教育、财务分析等多个行业领域，灵活满足多场景的计算需求。

- **插件名称** | **组件说明** | **默认限流** | **计费说明**
- 计算器 | 当前仅支持接入Doubao-pro系列模型，增强数学计算能力。 | 1QPS
- 60RPM | 限时免费
- 注：当前计算器插件暂不支持与其他插件同时开启使用

## 能力范围
计算器插件具有数值计算、方程和不等式求解、数学表达式化简、函数求导等计算能力，可支持广泛的数学应用范围

- **插件能力** | **相关应用范围**
- 数值计算 | 基础数值计算，适用于广泛数学计算问题
- 方程和不等式求解 | 适用于数学应用题中的求解代数方程（线性方程组、非线性方程）和不等式等问题
- 数学表达式化简 | 可在表达式上进行展开、因式分解、合并同类项和其它代数变换操作，以返回得到最简形式，适用于方程求解等场景
- 函数求导 | 解决高等数学中的函数求导与应用问题

## 功能优势

* **增强计算能力**：赋予模型可靠的数学能力，支持快速的计算响应。
* **灵活集成应用：** 接入大模型可广泛应用于多种计算场景。

## 应用场景
该插件具有广泛适用性：包括但不限于学习教育、财务分析等多个领域，满足多样化的数学计算需求。

* 学习教育：在教育场景下，通过接入模型提供高效的数学计算工具，辅助教学与学习。
* 财务分析：帮助财务人员进行快速、准确的财务计算和分析，提升工作效率。
* 其他数学计算场景：适用于需要数学计算支持的其他领域，如工程设计、市场预测等。

## 基本用法

* 控制台体验：用户可通过零代码应用创建的可视化界面进行接入与使用，详情请查看 [零代码应用操作指南](https://www.volcengine.com/docs/82379/1267885)。
* API 调用：支持通过 BotChatCompletions 接口调用已添加计算器插件的零代码应用，详情请查看 [Bot API 文档](https://www.volcengine.com/docs/82379/1285207) 与 [Bot SDK 文档](https://www.volcengine.com/docs/82379/1330201)。
