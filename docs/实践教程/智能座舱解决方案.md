# 智能座舱解决方案
一、**智能座舱解决方案是以豆包大模型为核心的座舱智能助理，** **是上一代座舱语音助手的升级，旨在通过大模型深度理解用户需求，提供更佳的座舱体验。目前核心的场景是：**

* 通过豆包大模型精准判断用户意图，从而链接座舱的全面智能服务：娱乐、车控、导航、本地生活等场景
* 基于豆包大模型打造的风格化车机人设助手，提升座舱中闲聊、陪伴的体验

**三、我们的优势：**

* 出色的模型能力：不同模型满足不同场景的需求，function call、结构化输出让模型效果更进一步
* 丰富的配套生态：基于头条搜索、抖音视频、本地生活、景点、天气、POI等插件生态提供全方位服务
* 高稳定性和性能：自动扩缩容智能调度，流量波动无须担心，保障座舱中良好的用户体验
