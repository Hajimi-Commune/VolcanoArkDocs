# 查看知识库列表
# 操作步骤

在知识库页面可查看已创建的知识库列表。
   ![alt](https://portal.volccdn.com/obj/volcfe/cloud-universal-doc/upload_8321abf4ef2787b8413beb5d216c5354.png)
   其中，知识库列表的各参数含义如下：
| **参数名称** | **参数说明** |
| --- | --- |
| 名称 | 知识库名称。 |
| 创建人 | 知识库的创建人。 |
| 创建时间 | 知识库的创建时间。 |
| 文档数量 | 该知识库中已导入的文档数量。 |
| 切片数量 | 该知识库中所有切片数量。 |
| 向量模型 | 该知识库使用的向量模型。 |
| 操作 | - **导入文档**：点击「导入文档」后会进入导入文档页面。 |\
|| - **编辑**：单击「编辑」后会进入编辑知识库页面，支持修改知识库描述和CPU。 |\
|| - **删除**：只能删除没有索引的数据集，单击「删除」后，弹窗提示确认是否删除，单击确定即可删除数据集。 |
