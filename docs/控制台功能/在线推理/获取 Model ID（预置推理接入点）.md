# 获取 Model ID（预置推理接入点） 
您可以基于 **Model ID** （模型 ID）快速调用已开通的大模型，系统将自动完成预置推理接入点的关联，无需人工管理。
## 获取 Model ID 
您可以通过以下两种途径获取 **Model ID**：

1. 参见 [模型列表](/docs/82379/1330310)，查阅所有可用模型。
2. 访问 [模型广场](https://console.volcengine.com/ark/region:ark+cn-beijing/model)，在模型卡片详情页面获取。
