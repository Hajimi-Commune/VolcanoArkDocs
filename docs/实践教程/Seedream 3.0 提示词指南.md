# Seedream 3.0 提示词指南
本文介绍 Seedream 文生图 3.0  的提示词（prompt）使用技巧，帮助您快速上手图片创作，将创意转化为图片内容。
# 提示词秘籍

* 用**简单直接**的语言表达，在表达准确的基础上，短提示词也能发挥惊艳的效果

> 长提示词并不一定代表好的生图效果，自己输入的提示词可能会比ai生成的更有效哦
