# 知识库插件 数据结构
知识库插件返回的数据结构如下：
## KnowledgeBaseChunk

- 参数名称 | 类型 | 描述 | 示例值
- collection_name | String | Collection 名字 | -
- project | String | Collection 所在的 project name | -
- doc_id | String | 文档 id | -
- doc_name | String | 文档名字 | -
- doc_type | String | 文档类型 | -
- doc_title | String | 文档标题 | -
- chunk_id | String | 分块 id | -
- chunk_title | String | 分块标题 | -
