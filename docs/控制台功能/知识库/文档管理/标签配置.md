# 标签配置
# 操作步骤

1. 创建知识库时，对知识库内所需标签进行初始化定义。

:::tip
注意：知识库创建后不能再对标签进行编辑，请提前配置好您所需的标签名
:::
![图片](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/864e8ff9bb79498eb85f9b86339e0824~tplv-goo7wpa0wc-image.image)

2. 在知识库列表中，点击知识库卡片，即可进入知识库详情页面，在原始文档页面下的文档列表，点击【标签】打开标签弹窗进行编辑。

![图片](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/5c331d5f70324d509c570e05ac7f2258~tplv-goo7wpa0wc-image.image)

3. 标签配置后，该文档内容即可在检索时通过对应标签选项进行过滤

![图片](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/a6bead0546f747f89abab0f9c24b50ee~tplv-goo7wpa0wc-image.image)

4. 在检索测试时，可以通过编辑过滤条件，对知识库内文档进行过滤，仅在满足条件的文档范围内进行检索

![图片](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/74cc30481e0b45bf88c43737e746a49f~tplv-goo7wpa0wc-image.image)
