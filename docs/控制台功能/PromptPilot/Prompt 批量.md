# Prompt 批量
在PromptPilot独立站版本中，若用户已有批量数据集，可从「Prompt批量」模块进入，直接上传数据集。用户将跳过「调试」环节，直接进入「批量」流程开展Prompt优化迭代。调试、批量、智能优化不同流程的操作方式与 [Prompt调优](https://www.volcengine.com/docs/82379/1399497) 一致。
