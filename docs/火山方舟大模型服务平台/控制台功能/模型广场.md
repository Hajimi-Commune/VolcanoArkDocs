# 模型广场
[模型广场](https://console.volcengine.com/ark/region:ark+cn-beijing/model) 汇集了时下主流的大模型，用户可以自由地查看及体验模型在语言、图像等方面的先进能力。模型广场包含语言大模型、视觉大模型、语音大模型、向量化模型，后续还会支持更多大模型，敬请期待。

# 模型对比
模型对比页面支持以模型效果，模型能力，模型价格，tokens 限制和任务类型等维度进行模型比较，分数越高，模型能力越强。

# 模型视图
模型广场支持 **列表视图** 和 **厂商视图** 两种模型展示方式，您可以在页面右上角进行视图的切换。
「列表视图」为您提供了全面而灵活的模型浏览方式。您可以自由探索不同类型的模型，并根据模型类别、多模态、文本、视觉、语音、向量等属性进行筛选。同时，支持按照模型能力，如函数调用、体验能力、精调能力、结构化输出、会话缓存、前缀缓存，以及上下文长度和更新时间等多个维度，快速定位符合您需求的模型。
「厂商视图」向您展示了不同模型厂商提供的各类模型，为您提供全新的模型浏览和选择体验。您可以快速找到感兴趣的厂商以及相关模型，清晰地了解模型的核心特点和能力，快速找到符合要求的模型。
「厂商视图」还汇集了豆包家族的主流大模型，提供高质量的语言模型与多模态模型矩阵。涵盖Doubao-pro、Doubao-lite系列语言大模型，Doubao-embedding向量模型，Doubao-声音复刻、Doubao-语音合成等语音大模型，Doubao-文生图、Doubao-图生图等视觉大模型。

# 模型卡片
模型广场中的每个模型都有专属的模型卡片，模型卡片展示名称、简介、提供方、是否可体验以及应用场景，通过单击模型卡片进入详情页后还能查看更详细的说明文档，通过阅读说明文档用户可以判断该模型是否对自身有帮助。
