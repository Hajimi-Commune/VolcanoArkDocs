# 安心体验模式细则说明
# 什么是安心体验模式？
安心体验模式是火山方舟平台为您提供的体验模式，可以保障您免费体验模型推理服务，又不产生费用。在安心体验模式下，您调用推理 API 时仅消耗平台赠送的 50w token 免费额度，接近免费额度后服务将暂停，避免产生额外费用。
:::warning
免费额度仅适用于抵扣模型推理消耗的 token（50w 免费 token），不能抵扣使用各类插件、知识库等产生的费用。因此，即使您开启安心体验模式，如果使用了联网插件、知识库等内容，这部分费用是无法抵扣的。
:::
# 模式说明
安心体验模式为账号维度的配置，适用于账号下所有支持安心体验模式的模型。

* **安心体验模式开启时**：调用模型在线推理时仅消耗免费额度，超出免费额度时服务将自动暂停。
* **安心体验模式关闭时**：调用模型在线推理时优先消耗免费额度，超出免费额度后按实际 Token 消耗计费

:::warning
* 如您在服务暂停后希望继续调用，需关闭安心体验模式；注意关闭后，将无法再次开启。
* 默认状态下安心体验模式关闭，如有需要请参见[功能入口](/docs/82379/1465347#34d3c92e)开启。
:::
# 生效范围

* **对象支持**：**未开通过模型服务**，且没开&关过安心体验模式的账号（包括实名认证通过的个人账号和企业账号）可开启。
* 支持的模型：除向量化、视频生成、同声传译模型。
* 功能支持：
   * 可避免付费：在线推理按 Token 后付费；
   * 不可避免付费：使用各类插件、知识库等产生的费用。
* 注意：如需购买 TPM 保障包、模型单元等，或开启推理缓存，需提前关闭安心体验模式。

# 功能入口
## 开启模式
入口1：进入[开通管理页面](https://console.volcengine.com/ark/region:ark+cn-beijing/openManagement?LLM=%7B%7D&OpenTokenDrawer=false)，在模型列表上方 “安心体验模式” 横幅中点击开关即可开启
入口2：（快速接入相关页面中）在开通模型时勾选开启安心体验模式，即可随模型一键开通
## 关闭模式
进入**开通管理页面**，在模型列表上方 “安心体验模式” 横幅中点击开关即可关闭
## 查看模型安心体验状态
进入**开通管理页面**，在模型列表中可查看对应模型是否处于安心体验模式下（“免费推理额度”列会有“安心体验”图标）
账号处于安心体验模式下时，模型消耗完平台赠送的 50w tokens 免费推理额度时对应模型服务将自动暂停（“状态”列会有“服务暂停”图标），如需继续使用请关闭安心体验模式
在推理 API 接入入口处，也可查看安心体验模式状态（快捷 API 接入、在线推理、体验中心等）

# 常见问题解答（Q&A）
### Q：安心体验模式开启后，服务何时会暂停？
A：当平台赠送的 50w Tokens用尽时，服务将自动暂停，您可在开通管理页面的模型列表中查看详情。
### Q：关闭安心体验模式后，还能重新开启吗？
A：由于安心体验模式仅针对新用户体验免费额度，在关闭后无法重新开启，请谨慎操作。
### Q：哪些模型不支持安心体验模式？
A：Embedding 模型、视频生成模型、同声传译模型等暂不支持。具体支持的模型您可在开启了安心体验模式后，在开通管理页的模型列表中查看，支持的模型会显示“安心体验”标识。
### Q：免费额度用尽后，如何继续使用服务？
A：您可以前往开通管理页面**关闭安心体验模式**，之后超出免费额度的部分将按实际 Token 消耗计费。
### Q：为什么开通管理的模型列表中显示模型免费推理额度还有余量，但是服务已经被暂停了？
A：如您开启了安心体验模式，模型服务会在您实际消耗完平台赠送的 50w tokens 免费推理额度时暂停，而开通管理页面“免费推理额度”为账号下对应模型免费资源包总量，且费用中心的数据更新时间与实际调用会存在小时级延迟；因此，安心体验模式下是否用完了免费额度请以 API 返回信息和模型列表的状态为准
### Q：为什么我之前可以看到安心体验模式，但现在看不到对应入口了？
A：安心体验模式是针对未开通过模型服务的新用户更方便体验免费额度设计的，在开通任一模型服务、且开通模型服务时未选择开启安心体验模式后，将不满足账户开启安心体验模式的条件
