# Token 计算器
Token 计算器是一款在线工具，可以为平台上的文本、多模态模型精确预估 Token 消耗。它能帮助你在正式调用模型 API 前，清晰地了解各项任务的资源占用情况，从而有效评估和管理成本。本文将详细介绍 Token 计算器的各项功能，并指导你如何为不同类型的模型计算 Token。
### 关键特性

* **独立的访问入口**：提供独立的 [Token 计算器页面](https://console.volcengine.com/ark/region:ark+cn-beijing/tokenCalculator?projectName=default) 方便你快速访问。
* **广泛的模型支持**：能够根据所选模型，包括文本生成模型、图片内容理解模型（Visual Language Model, VLM）等自动应用对应的分词器和计算逻辑，确保计算结果的准确性。
* **优化的交互体验**：整体页面布局和交互流程经过优化，提升了操作的流畅性和易用性。

## 操作步骤
### 计算文本模型 Token
本节介绍如何为纯文本模型计算 Token。

1. 在 **Token 计算器** 页面，通过顶部的分类筛选器或直接在模型列表中，选择你要计算的文本模型。
2. 在 **文本输入** 输入框中，输入或粘贴待计算的文本内容。
3. 在页面右侧的计算结果区域，查看文本的 Token 消耗，以及内容分词情况。

### 计算多模态模型 (VLM) Token
本节介绍如何为多模态模型（VLM）计算文本和图片的混合 Token。

1. 在模型选择区域，选择一级分类**图片内容理解**，然后选择具体的 VLM 模型。
2. 根据你的需求，输入图片及文本信息。

- 配置项 | 说明
- **图片输入** | 必选。你可以通过以下任一方式提供图片信息：
- * **上传图片**：单击上传组件，选择本地图片。系统将自动读取并填充图片的宽度和高度。
- * **手动填写**：直接在 **图像宽度** 和 **图像高度** 输入框中填写图片的尺寸信息。
- **文本输入** | 可选。在 **分析文本** 输入框中输入与图片相关的文本内容。
- **图片理解精度** | 特定模型需要配置。例如，当选择模型为 Doubao-1.5-vision-pro-32k 时，你需要选择图片理解精度。

> **说明**：你可以随时修改已上传的图片或手动填写的宽高，系统会基于最新输入进行计算。

3. 在右侧区域查看分项计算结果，包括 **图片 Tokens**、**文本 Tokens** 和 **Total Tokens**（总消耗）。
