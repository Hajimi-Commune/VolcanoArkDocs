# Canvas 简介
Canvas（画布）功能提供了一个独立的工作空间，使您可以与 AI 并肩合作，在同一画布上进行内容的创作、预览与优化，能够帮您显著提高写作、编程等复杂任务的完成质量和效率。
启用 Canvas 功能后，在对话窗口涉及创意写作与代码时，系统会自动打开 Canvas 画布展示，您还可直接于画布上编辑文本和代码内容。
:::tip
为获得更佳的 Canvas 体验效果，建议使用深度思考模型：Doubao-1.5-thinking-vision-pro，Doubao-1.5-thingking-pro，DeepSeek-R1。
:::
