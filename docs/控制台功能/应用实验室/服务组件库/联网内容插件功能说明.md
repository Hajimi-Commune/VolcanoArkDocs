# 联网内容插件功能说明
# 功能简介
尽管大语言模型具备强大的语言理解与生成能力，但它们的训练数据通常是固定的通用语料库，这意味着大模型在训练后无法实时更新信息，并且对于某些特定领域的问题，可能缺少足够的专业知识。
联网内容插件，支持实时搜索互联网公开域网页资源和字节系的图文、视频资源，可以帮助大语言模型获取最新的、专业的信息，提升大模型回答效果。对于字节系的图文和视频资源，支持查看大模型引用的原文链接或视频，视频化解读用户问题，升级交互体验。
联网内容插件支持的搜索资源：

|---|---|---|---|---|
| | | | | | \
|资源说明 |涵盖各领域丰富的互联网公开可搜索的网页 |头条app提供的tob合规的图文资讯 |抖音app提供的tob合规的百科词条 |墨迹天气提供的晴雨、温度、湿度、空气质量、风向风力等城市天气信息 |

* 默认限流：5QPS （账号维度）
* **注意：开启此插件时，Function Calling 功能不生效。**

# 功能优势

* **多模态交互体验：** 支持联网信息的快速响应以及文本、图片、百科、视频等多模态内容呈现，提供流畅丰富的智能交互体验
* **优质丰富数据：** 丰富多样的数据源，包括并不限于字节系长期沉淀积累的优质数据，灵活配置可支撑各行业落地应用
* **实时联网信源：** 实时补充与更新联网信源，支持高时效问题回答，有效弥补大模型幻觉、知识更新不及时的缺陷
* **领先的搜索框架：** 高准确率的意图识别，联合问题改写与召回策略，可精准捕获用户意图并高效执行特定场景任务

# 基本用法
## 创建带联网插件的零代码应用
您可通过控制台可视化的创建 **零代码应用**，详情请查看 [零代码应用操作指南](https://www.volcengine.com/docs/82379/1267885)。
创建完成后，您可以通过以下 2 种方式使用带联网插件的应用：

* 控制台体验：直接在[我的应用](https://console.volcengine.com/ark/region:ark+cn-beijing/assistant)的可视化界面中，通过控制台体验应用。
* API 调用：在[我的应用](https://console.volcengine.com/ark/region:ark+cn-beijing/assistant)获取应用ID，使用接口调用。接口详情请查看 [Bot API 文档](https://www.volcengine.com/docs/82379/1285207) 与 [Bot SDK 文档](https://www.volcengine.com/docs/82379/1330201)。

## 通过 Responses API 调用联网搜索工具
您可以通过 Responses API 来调用基础联网搜索工具获取公开域互联网信息（如新闻、商品、天气等），详情请参考[联网搜索 Web Search ](/docs/82379/1756990)。

**curl**

```Bash
curl --location 'https://ark.cn-beijing.volces.com/api/v3/responses' \\
--header "Authorization: Bearer $ARK_API_KEY" \\
--header 'Content-Type: application/json' \\
--data '{
    "model": "doubao-seed-1-6-250615",
    "stream": true,
    "tools": [
        {"type": "web_search"}
    ],
    "input": [
        {
            "role": "user",
            "content": [
                {
                    "type": "input_text",
                    "text": "今天有什么热点新闻"
                }
            ]
        }
    ]
}'
```

**兼容 OpenAI SDK**

```Python
from openai import OpenAI
import os

# 从环境变量中获取您的API KEY，配置方法见：https://www.volcengine.com/docs/82379/1399008
api_key = os.getenv('ARK_API_KEY')

client = OpenAI(
    base_url='https://ark.cn-beijing.volces.com/api/v3',
    api_key=api_key
)

tools = [{
    "type": "web_search",
}]

# 创建一个对话请求
response = client.responses.create(
    model="doubao-seed-1-6-250615",
    input=[{"role": "user", "content": "北京的天气怎么样？"}],
    tools=tools,
)

print(response)
```
