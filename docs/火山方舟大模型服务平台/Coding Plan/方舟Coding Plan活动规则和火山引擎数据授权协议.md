# 方舟Coding Plan活动规则和火山引擎数据授权协议
## 火山**方舟Coding Plan 活动规则**
火山引擎邀请客户（“您”）参与**Coding Plan**计划（“本计划”），本计划旨在与您一起共建更优质的模型服务。
### Coding Plan 权益说明
#### 活动时间
2025.11.11 - 2026.05.11
#### 活动对象
所有完成认证（包含个人认证和企业认证）的火山引擎用户，每个实名认证主体可享受一次首月优惠价。
#### 活动规则

1. 购买限制：单个账号最多可以购买 6 个月。
2. 购买不同套餐：
   * 同一账号只能购买一个类型的套餐，请根据使用需求选择合适的套餐。
   * 同一时间只有一个套餐生效，当前套餐到期后，下一个套餐会自动生效。
3. 回购规则：如果当前套餐在有效期内，则不支持再次购买。
4. 续费规则：
   * 在下单页选择自动续费：即连续包月，每次固定续费 1 个月，到期会自动扣除下一个月的套餐费用。
   * 在下单页没有选择自动续费：套餐到期后会短信通知用户，用户需要重新订阅套餐。
   * 在下单页手动续费：再次购买已订阅的套餐，将根据购买的月数延长订阅的有效期。
5. 升配规则：
   * 有效期不变，补齐要升配套餐的差价后即可升配。
   * 仅支持升配，不支持降配。
6. 退款规则：请注意套餐购买后不支持退款，您可以在当前周期内使用服务。建议您根据使用需求选择合适的订阅套餐和周期。

#### 首次订阅优惠规则

* 订阅单月套餐：享受首月优惠价。
* 订阅多月套餐：仅有 1 个套餐享受首月优惠价，其余套餐原价。

本计划仅支持同一用户的唯一个账号参与，即如您同一主体下有多个账户或多个账号，这些账户或账号不可同时申请参与本计划。
**您在活动期内调用火山方舟指定模型服务期间的调用数据，即因调用模型而输入和模型生成的内容，包括但不限于所有输入的文本、声音、图形、图片、图像、视频、软件或任何其他类似信息（以下称“AI Coding数据”）将授权提供给火山引擎用于火山引擎及其关联公司的模型和算法优化。**
### 数据授权说明

**您理解并确认，火山引擎可永久使用您在活动期内已经授权给火山引擎的AI Coding数据。如您已参与本计划，但在参与过程中希望撤回授权，您可选择立即停止使用【Coding Plan 订阅包月套餐】服务。您理解并同意，【Coding Plan 订阅包月套餐】不支持退订退款，即因您放弃继续使用已订购的【Coding Plan 订阅包月套餐】，您将无法获得退款。**
