# 续订知识库
仅当下单【预付费】模式知识库，在收到资源到期提醒时，可参考当前文档对知识库资源进行续订。
知识库预付费资源分为：按月抵扣的计算&离线存储资源，按用量抵扣的大模型资源包，两类资源可独立订阅按需续订
**1、按月抵扣的计算&离线存储资源**
在控制台右上角点击【续费管理】进入到资源续费页面

![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/37e9b1c8600e41d287050a4c0c8dfb63~tplv-goo7wpa0wc-image.image =2900x)
在【产品】筛选【火山方舟大模型服务平台-知识库】，在筛选结果中会出现两个实例，分别是计算资源 和 离线存储资源。点击【续费】即可对两个资源进行续费操作。
⚠️注意：为保证知识库服务的正常使用，建议两个资源设置相同的到期时间
![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/c21d4b5945ed4796bf12ed19b690a2f0~tplv-goo7wpa0wc-image.image =2922x)
**2、大模型资源包单独增购**
点击 [增购链接](https://console.volcengine.com/common-buy/VDB_KnowledgeBase%7C%7C7405069681198911788) 进入大模型资源包增购页面，模型 tokens 均按实际用量抵扣，有效期 12 个月
