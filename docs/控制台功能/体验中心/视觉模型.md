# 视觉模型
# 进入体验中心
可从以下位置进入体验中心：

* 在控制台左侧导航栏中，点击 [体验中心](https://console.volcengine.com/ark/region:ark+cn-beijing/experience/chat)。
* 在 **模型广场** 中打开模型详情页，点击 **立即体验** 进入体验中心。

# 试用模型
视觉模型包括视频生成、图片生成和 3D 生成模型。
下文以图片生成模型为例，介绍试用方法。

1. 在体验中心界面左上方展开模型列表，选择需要试用的模型、模型版本/推理接入点。
   ![Image](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/e1c7887c6a61485bb4348962f245392b~tplv-goo7wpa0wc-image.image)
2. 打开 **模型参数** 面板，选择是否生成组图，并配置分辨率等参数。
3. 在底部输入框输入提示信息，包括文字描述、参考图片，点击右下角按钮即可开始生成。也可使用模板快速体验。
