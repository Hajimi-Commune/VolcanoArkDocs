# 控制台-推理接入点安全监控总览
接入点安全监控页面为用户提供全局化的安全状态监控和便捷的风险总览，帮助用户快速识别并定位账号中所有接入点的潜在安全风险。页面包含**安全防护能力**、**安全风险事件**以及**存在风险的接入点**。
# 入口
在用量统计页面中，点击页面中间的 **接入点安全监控** tab，可查看这个账号下所有接入点的安全审计数据。

# 安全防护能力看板
上方的「安全防护能力看板」为您展示平台的安全防护能力及开启状态。其中，**vArmor容器沙箱**、**网络层传输加密**和**网络隔离**为默认开启。

| | | \
|**指标名称** |**含义** |
|---|---|
| | | \
|**应用层会话加密** |显示最近7天内未开启应用层会话加密的接入点数量及详情 |
| | | \
|**vArmor容器沙箱** |统计vArmor拦截的风险次数，确保任务执行过程中的安全威胁被防护 |
| | | \
|**网络层传输加密** |展示网络层传输加密是否开启的状态 |
| | | \
|**网络隔离** |通过任务级网络隔离策略，保障同一网络中的不同任务互相隔离，防止直接通信，提高网络安全性 |

<div style="text-align: center"><img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/73e2b634e8414d31a6546d1cbd5cd246~tplv-goo7wpa0wc-image.image" width="3483px" /></div>

看板中显示账号内近7天未开启应用层会话加密的接入点数量，鼠标悬停可查看未开启加密的接入点详情，您可以根据[指引](/docs/82379/1389905)前往对应接入点自行开启。
<div style="text-align: center"><img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/e656458107f54ec38fdf75f05e355104~tplv-goo7wpa0wc-image.image" width="3518px" /></div>

看板中安全能力名称旁带有“**i**”图标时，将鼠标悬停即可查看详细说明，帮您更好地理解对应的安全能力。
<div style="text-align: center"><img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/5ae7c4e0623f4e71bca5b8d44ca93e26~tplv-goo7wpa0wc-image.image" width="1176px" /></div>

# 安全风险事件看板
<div style="text-align: center"><img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/f7cc5ceb2f6b47a191904d57b6cafe88~tplv-goo7wpa0wc-image.image" width="3522px" /></div>

安全风险事件模块分为**「安全风险事件总览」**和**「安全风险事件趋势图」**

| | | \
|**指标名称** |**含义** |
|---|---|
| | | \
|**沙箱异常连接** |统计近30天内沙箱对外的异常网络连接次数，反映沙箱是否有主动向外发送数据行为 |
| | | \
|**沙箱异常登录** |统计近30天内通过远程或本地方式登录沙箱的行为次数，帮助识别潜在的窃取沙箱内数据的操作 |
| | | \
|**沙箱容器逃逸** |统计近30天内超越沙箱权限的执行命令数量，监控是否有未经允许的越权操作 |

中间的**「安全风险事件总览」**为您展示指定时间内所有接入点的安全风险信息，包括沙箱异常连接、沙箱异常登录和沙箱容器逃逸的数量，帮助您直观了解所有接入点的安全风险状态。如果数字为0，代表指定时间内没有发生该项风险。
<div style="text-align: center"><img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/f34d48811e044c768e25d56f08e22f9c~tplv-goo7wpa0wc-image.image" width="1176px" /></div>

右侧的**「安全风险事件趋势图」**展示了所选时间范围内所有接入点的安全风险数量及变化趋势，包括**全部安全风险**、**沙箱异常连接**、**沙箱异常登录**和**沙箱容器逃逸**。默认显示所有趋势，您可以通过点击对应的趋势按钮隐藏或显示特定的风险趋势。
<div style="text-align: center"><img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/7ef3d0c7f2734569ac6eb83d660361f0~tplv-goo7wpa0wc-image.image" width="1164px" /></div>

# 存在风险的接入点看板
下方的**「存在风险的接入点看板」**展示了风险接入点总数及风险接入点详情。
<div style="text-align: center"><img src="https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/b421d324dda24fcaa261d7053940d88b~tplv-goo7wpa0wc-image.image" width="3489px" /></div>

左侧的**「风险接入点总数」**为您展示指定时间内存在风险的接入点总数**，**如果数字为0，代表指定时间内所有接入点没有发生风险。
右侧的**「风险接入点详情」**为您展示指定时间内存在风险的接入点详情**，包括接入点名称、发生风险次数，**帮助您快速定位指定时间内发生风险的接入点。
点击查看详情，可跳转对应接入点的安全审计页面查看发生风险的详细信息。
